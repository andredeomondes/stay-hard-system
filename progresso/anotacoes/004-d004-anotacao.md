## O que aprendi:

### DNS (Domain Name System)
- Sistema que traduz nomes de domínio em endereços IP
- Exemplo: google.com → 142.250.217.78

### URL (Uniform Resource Locator)
- Endereço de recurso na web

### Estrutura de URL:
```
https://www.google.com:443/search?q=java&hl=pt-BR
│     │    │              │    │        │
│     │    │              │    │        └─ Query String
│     │    │              │    └─────────── Path
│     │    │              └────────────────── Porta
│     │    └──────────────────────────────── Domínio
│     └─────────────────────────────────────── Protocolo
└───────────────────────────────────────────── URL completa
```

### Portas mais comuns
| Porta | Serviço |
|-------|---------|
| 80 | HTTP |
| 443 | HTTPS |
| 8080 | HTTP alternativo (desenvolvimento) |

---

## Insights:

- DNS é como uma "agenda de telefones" da internet
- URL é o endereço completo de um recurso
- Porta 443 = HTTPS (seguro)
- Porta 80 = HTTP (não seguro)
