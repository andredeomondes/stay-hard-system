Este é um roadmap de 209 dias para se tornar **Desenvolvedor Backend Java Full-Stack**.

- **Messaging**: Kafka, Redis

## 🎓 REGRAS PARA CRIAR DIÁRIOS

### Formato de cada diário (docs/daily/XX-dXX.md):

```markdown
# Dia XX — [Título]

## Conceito do Dia
[Explicação breve do conceito]

## Onde Estudar
- [Links para recursos]

## Checklist de Código
- [ ] Tarefa 1
- [ ] Tarefa 2

## Código/Exemplo
[Código relevante]

## Aplicar no Stay Hard
[Como aplicar no projeto]

## Comandos Essenciais
```bash
# comandos
```

## Verificação
- [ ] Checklist final
```

### Princípios:
1. **Específico**: Não genérico, com exemplos concretos
2. **Prático**: Código executável
3. **Progressivo**: Baseado em dias anteriores
4. **Aplicável**: Sempre连接到 projeto Stay Hard
