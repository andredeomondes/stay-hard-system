# Dia 193 — GraphQL: Subscriptions

## Conceito do Dia

**Subscriptions** entregam dados em tempo real via WebSocket.

---

## Schema

```graphql
type Subscription {
    tarefaCriada: Tarefa!
    tarefaAtualizada(id: ID!): Tarefa!
    tarefaDeletada: ID!
}
```

## Publisher

```java
@Service
public class TarefaSubscriptionPublisher {
    
    private final ApplicationEventPublisher<TarefaEvent> publisher;
    
    public void publish(TarefaEvent event) {
        publisher.publishEvent(event);
    }
}

@Service
public class TarefaService {
    
    private final TarefaSubscriptionPublisher publisher;
    
    public Tarefa criar(CriarTarefaRequest request) {
        Tarefa tarefa = repository.save(request);
        publisher.publish(new TarefaCriadaEvent(tarefa));
        return tarefa;
    }
}
```

## Subscriber

```java
@Component
public class TarefaGraphQLSubscription {
    
    private final TarefaSubscriptionPublisher publisher;
    
    @SubscriptionMapping
    public Publisher<Tarefa> tarefaCriada() {
        return PublisherFactory.createPublisher("tarefa-criada");
    }
}
```

## Client (React)

```javascript
import { useSubscription, useMutation } from '@apollo/client';

const TAREFA_CRIADA_SUBSCRIPTION = gql`
  subscription OnTarefaCriada {
    tarefaCriada {
      id
      nome
      categoria
    }
  }
`;

function TarefasList() {
  const { data, loading } = useSubscription(TAREFA_CRIADA_SUBSCRIPTION);
  
  return (
    <div>
      {data?.tarefaCriada && (
        <NewTarefaAlert tarefa={data.tarefaCriada} />
      )}
      {/* lista existente */}
    </div>
  );
}
```
