### Definição em 1 frase:

### Sintaxe:
```
(params) -> expression
(params) -> { statements }
```

### 📁 Exercício 1: Lambda Básico

```java
package com.estudo;

import java.util.*;

public class LambdaDemo {
    public static void main(String[] args) {
        // Sem lambda (classe anônima)
        Comparator<String> semLambda = new Comparator<String>() {
            @Override
            public int compare(String a, String b) {
                return a.length() - b.length();
            }
        };
        
        // Com lambda
        Comparator<String> comLambda = (a, b) -> a.length() - b.length();
        
        // Lista de nomes
        List<String> nomes = List.of("Carlos", "Ana", "Bob", "Diana");
        
        // Ordenar com lambda
        Collections.sort(nomes, (a, b) -> a.length() - b.length());
        System.out.println(nomes); // [Bob, Ana, Carlos, Diana]
        
        // Ou com stream
        List<String> ordenado = nomes.stream()
            .sorted((a, b) -> a.length() - b.length())
            .toList();
    }
}
```

**Checklist:**
- [ ] Classe anônima vs Lambda
- [ ] Comparator com lambda
- [ ] sort com lambda

### 📁 Exercício 2: Lambdas com Streams

```java
List<Integer> numeros = List.of(1, 2, 3, 4, 5);

// Lambda completo
numeros.stream()
    .filter(n -> {
        boolean par = n % 2 == 0;
        return par;
    })
    .map(n -> {
        int dobrado = n * 2;
        return dobrado;
    })
    .forEach(n -> System.out.println(n));

// Lambda simplificado (1 linha)
numeros.stream()
    .filter(n -> n % 2 == 0)
    .map(n -> n * 2)
    .forEach(n -> System.out.println(n));
```

**Checklist:**
- [ ] filter com lambda
- [ ] map com lambda
- [ ] forEach com lambda

### 📁 Exercício 3: Method Reference

```java
List<String> nomes = List.of("André", "Maria", "João");

// Lambda
nomes.forEach(nome -> System.out.println(nome));

// Method Reference (mais limpo!)
nomes.forEach(System.out::println);

// Exemplo com map
List<String> maiusculos = nomes.stream()
    .map(nome -> nome.toUpperCase())  // Lambda
    .toList();

List<String> maiusculos2 = nomes.stream()
    .map(String::toUpperCase)          // Method Reference
    .toList();

// Exemplos de Method Reference
// Instância::método
nomes.forEach(String::toUpperCase);

// Classe::método estático
List<Integer> nums = List.of(3, 1, 2);
nums.stream().sorted(Collections::reverseOrder);

// Classe::novo (construtor)
Supplier<ArrayList<String>> supplier = ArrayList::new;
```

**Checklist:**
- [ ] System.out::println
- [ ] String::toUpperCase
- [ ] Collections::reverseOrder

### 📁 Exercício 4: Predicate, Consumer, Function

```java
import java.util.function.*;

// Predicate: recebe T, retorna boolean
Predicate<String> naoVazio = s -> !s.isEmpty();
Predicate<String> temMaisDe3 = s -> s.length() > 3;

System.out.println(naoVazio.test("Olá")); // true
System.out.println(temMaisDe3.test("Oi")); // false

// Combinar predicates
Predicate<String> valido = naoVazio.and(temMaisDe3);
System.out.println(valido.test("André")); // true

// Consumer: recebe T, não retorna nada
Consumer<String> printUpperCase = s -> System.out.println(s.toUpperCase());
printUpperCase.accept("Olá"); // OLÁ

// Function: recebe T, retorna R
Function<String, Integer> length = String::length;
System.out.println(length.apply("André")); // 5

// Supplier: não recebe nada, retorna T
Supplier<List<String>> listaSupplier = () -> List.of("A", "B");
List<String> lista = listaSupplier.get(); // [A, B]
```

**Checklist:**

- [ ] Supplier
