## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**Interfaces — Contratos entre classes**

### Definição em 1 frase:
Interface define O QUE uma classe deve fazer, sem dizer COMO.

### Exemplo:
```
interface Pagavel
├── receberPagamento()
├── calcularJuros()
└── gerarRecibo()

class Fatura implements Pagavel
class Contrato implements Pagavel
class NotaFiscal implements Pagavel
```

### 📁 Exercício 1: interface

```java
// Interface (contrato)
public interface Autenticavel {
    boolean autenticar(String usuario, String senha);
    void fazerLogout();
}

// Implementações
public class LoginFormulario implements Autenticavel {
    
    @Override
    public boolean autenticar(String usuario, String senha) {
        // Lógica específica do formulário
        return usuario.equals("admin") && senha.equals("123");
    }
    
    @Override
    public void fazerLogout() {
        // Limpar sessão
    }
}

public class LoginOAuth implements Autenticavel {
    
    @Override
    public boolean autenticar(String usuario, String senha) {
        // Lógica OAuth
        return true;
    }
    
    @Override
    public void fazerLogout() {
        // Invalidar token
    }
}
```

**Uso:**
```java
// Polimorfismo com interfaces
Autenticavel auth = new LoginFormulario();
if (auth.autenticar("admin", "123")) {
    auth.fazerLogout();
}

// Lista de autenticáveis
List<Autenticavel> metodos = List.of(
    new LoginFormulario(),
    new LoginOAuth()
);
```

**Checklist:**
- [ ] interface
- [ ] implements

### 📁 Exercício 3: Interfaces múltiplas

```java
public interface Vendivel {
    double getPreco();
}

public interface Descontavel {
    double aplicarDesconto(double percentual);
}

public interface Transportavel {
    double getPeso();
}

// Classe pode implementar várias interfaces
public class Produto implements Vendivel, Descontavel, Transportavel {
    
    @Override
    public double getPreco() { return 100.0; }
    
    @Override
    public double aplicarDesconto(double pct) {
        return getPreco() * (1 - pct);
    }
    
    @Override
    public double getPeso() { return 0.5; }
}
```
