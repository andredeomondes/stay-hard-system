## Código: Builder para Tarefa

```java
// ANTES: Telescoping Constructor (horrível)
class Tarefa {
    private String nome;
    private String descricao;
    private Prioridade prioridade;
    private Categoria categoria;
    private LocalDateTime prazo;
    private boolean concluida;
    
    Tarefa(String nome) { this.nome = nome; }
    Tarefa(String nome, String descricao) { ... }
    Tarefa(String nome, String descricao, Prioridade p) { ... }
    // 2^n combinações possíveis!
}

// DEPOIS: Builder
class Tarefa {
    private final String nome;          // required
    private String descricao;           // optional
    private Prioridade prioridade;      // optional
    private Categoria categoria;        // optional
    private LocalDateTime prazo;        // optional
    private boolean concluida;          // optional
    
    private Tarefa(Builder builder) {
        this.nome = builder.nome;
        this.descricao = builder.descricao;
        this.prioridade = builder.prioridade;
        this.categoria = builder.categoria;
        this.prazo = builder.prazo;
        this.concluida = builder.concluida;
    }
    
    static class Builder {
        private final String nome;
        private String descricao = "";
        private Prioridade prioridade = Prioridade.MEDIA;
        private Categoria categoria;
        private LocalDateTime prazo;
        private boolean concluida = false;
        
        Builder(String nome) {
            this.nome = nome;
        }
        
        Builder descricao(String desc) {
            this.descricao = desc;
            return this;
        }
        
        Builder prioridade(Prioridade p) {
            this.prioridade = p;
            return this;
        }
        
        Builder prazo(LocalDateTime dt) {
            this.prazo = dt;
            return this;
        }
        
        Tarefa build() {
            return new Tarefa(this);
        }
    }
}

// Uso - muito mais legível!
Tarefa tarefa = new Tarefa.Builder("Estudar SOLID")
    .descricao("Assisti vídeos sobre SRP e OCP")
    .prioridade(Prioridade.ALTA)
    .prazo(LocalDateTime.now().plusDays(7))
    .build();
```

```java
// Com Lombok (mais limpo!)
@lombok.Builder
class TarefaLombok {
    private String nome;
    private String descricao;
    private Prioridade prioridade;
    private Categoria categoria;
}

// Uso
TarefaLombok tarefa = TarefaLombok.builder()
    .nome("Estudar")
    .prioridade(Prioridade.ALTA)
    .build();
```
