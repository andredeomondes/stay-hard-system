## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**SQL — Structured Query Language**

### Definição em 1 frase:
SQL é a linguagem para comunicar com bancos de dados relacionais.

### Comandos principais:
```
DDL (Data Definition Language)  → CREATE, ALTER, DROP
DML (Data Manipulation Language) → INSERT, UPDATE, DELETE, SELECT
DCL (Data Control Language) → GRANT, REVOKE
TCL (Transaction Control) → COMMIT, ROLLBACK
```

## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: CREATE TABLE

```sql
-- Criar banco de dados
CREATE DATABASE stayhard;

-- Usar banco
USE stayhard;

-- Criar tabela de usuários
CREATE TABLE usuarios (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nome VARCHAR(100) NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL,
    idade INT,
    ativo BOOLEAN DEFAULT TRUE,
    data_criacao TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

---

### 📁 Exercício 2: Tipos de Dados

```sql
-- Numéricos
INT              -- Inteiro (-2B a 2B)
BIGINT           -- Inteiro grande
DECIMAL(10,2)    -- Decimal (10 dígitos, 2 decimais)
BOOLEAN          -- verdadeiro/falso

-- Texto
VARCHAR(255)     -- Texto variável (até 255)
TEXT             -- Texto longo
CHAR(10)         -- Texto fixo (sempre 10)

-- Data/Hora
DATE             -- Data (2024-01-01)
TIME             -- Hora (12:30:00)
DATETIME         -- Data e hora
TIMESTAMP        -- Timestamp

-- Binário
BLOB            -- Arquivos (imagens, pdf)
```

---

### 📁 Exercício 3: ALTER TABLE

```sql
-- Adicionar coluna
ALTER TABLE usuarios ADD COLUMN telefone VARCHAR(20);

-- Modificar coluna
ALTER TABLE usuarios MODIFY COLUMN telefone VARCHAR(15);

-- Renomear coluna
ALTER TABLE usuarios CHANGE COLUMN telefone celular VARCHAR(15);

-- Deletar coluna
ALTER TABLE usuarios DROP COLUMN celular;

-- Adicionar constraint
ALTER TABLE usuarios ADD CONSTRAINT unique_email UNIQUE (email);
```

---

### 📁 Exercício 4: DROP TABLE

```sql
-- Deletar tabela (permanente!)
DROP TABLE usuarios;

-- Deletar se existir
DROP TABLE IF EXISTS usuarios;

-- Deletar banco
DROP DATABASE stayhard;
```

---

### 📁 Exercício 5: Constraints

```sql
CREATE TABLE exemplo (
    -- Primary Key
    id INT PRIMARY KEY,
    
    -- Não pode ser nulo
    nome VARCHAR(100) NOT NULL,
    
    -- Único
    email VARCHAR(255) UNIQUE,
    
    -- Valor padrão
    ativo BOOLEAN DEFAULT TRUE,
    
    -- Check constraint
    idade INT CHECK (idade >= 0 AND idade <= 150),
    
    -- Chave estrangeira
    empresa_id INT,
    FOREIGN KEY (empresa_id) REFERENCES empresas(id)
);
```
