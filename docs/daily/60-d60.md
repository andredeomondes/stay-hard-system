## Conceito do Dia

Consolidar tudo: crie uma **API REST simples** com Servlets que gerencia tarefas. Isso prepara o terreno para o Spring Boot.

---

## Mini-Projeto: Stay Hard API (Servlet Edition)

### Requisitos

- [ ] GET /api/tarefas - listar todas
- [ ] GET /api/tarefas/{id} - buscar por ID
- [ ] POST /api/tarefas - criar tarefa
- [ ] PUT /api/tarefas/{id} - atualizar tarefa
- [ ] DELETE /api/tarefas/{id} - deletar tarefa
- [ ] Filtro de CORS para permitir requisições do front-end
- [ ] Tratamento de exceções

## Tomcat Embarcado

```java
public class App {
    public static void main(String[] args) throws Exception {
        Tomcat tomcat = new Tomcat();
        tomcat.setPort(8080);
        tomcat.getConnector();
        
        Context ctx = tomcat.addContext("", null);
        
        // Adicionar Servlet
        Tomcat.addServlet(ctx, "tarefaServlet", new TarefaServlet());
        ctx.addServletMappingDecoded("/api/tarefas/*", "tarefaServlet");
        
        tomcat.start();
        tomcat.getServer().await();
    }
}
```
