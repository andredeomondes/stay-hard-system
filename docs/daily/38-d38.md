## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: Nomes ruins vs Bons

```java
// ❌ RUIM
int d;                    // o que é d?
String nm;               // nome?
boolean flg;             // flag?
Map<String, Object> m;   // mapa de quê?
List lst;                // lista de quê?
public void proc() { }   // processar o quê?
int x = usuarios.stream().filter(u -> u.getA() > 18).count();

// ✅ BOM
int idade;
String nomeUsuario;
boolean usuarioAtivo;
Map<String, Usuario> usuariosPorId;
List<Usuario> usuarios;
public void processarPedido() { }
int quantidadeDeAdultos = usuarios.stream()
    .filter(u -> u.getIdade() > 18)
    .count();
```
