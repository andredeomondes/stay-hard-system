## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: GROUP BY

```sql
-- Agrupar e agregar
SELECT 
    cidade,
    COUNT(*) AS total,
    AVG(idade) AS media_idade,
    MAX(idade) AS mais_velho,
    MIN(idade) AS mais_novo
FROM usuarios
GROUP BY cidade;

-- GROUP BY com WHERE
SELECT 
    categoria,
    COUNT(*) AS total_produtos,
    AVG(preco) AS preco_medio
FROM produtos
WHERE ativo = TRUE
GROUP BY categoria
HAVING COUNT(*) > 5;  -- Só categorias com +5 produtos
```

---

### 📁 Exercício 2: HAVING vs WHERE

```sql
-- WHERE: filtra ANTES do GROUP BY
SELECT cidade, COUNT(*)
FROM usuarios
WHERE idade >= 18
GROUP BY cidade;

-- HAVING: filtra DEPOIS do GROUP BY
SELECT cidade, COUNT(*) AS total
FROM usuarios
GROUP BY cidade
HAVING total > 10;  -- Só cidades com +10 usuários
```

---

### 📁 Exercício 3: Subquery no WHERE

```sql
-- Usuários mais velhos que a média
SELECT nome, idade
FROM usuarios
WHERE idade > (SELECT AVG(idade) FROM usuarios);

-- Usuários que têm pedido
SELECT nome
FROM usuarios
WHERE id IN (SELECT usuario_id FROM pedidos);

-- Usuários que NÃO têm pedido
SELECT nome
FROM usuarios
WHERE id NOT IN (SELECT usuario_id FROM pedidos);
```

---

### 📁 Exercício 4: Subquery no FROM

```sql
-- Ranking de usuários por quantidade de pedidos
SELECT 
    u.nome,
    stats.total_pedidos,
    stats.valor_total
FROM usuarios u
INNER JOIN (
    SELECT 
        usuario_id,
        COUNT(*) AS total_pedidos,
        SUM(preco) AS valor_total
    FROM pedidos
    GROUP BY usuario_id
) stats ON u.id = stats.usuario_id
ORDER BY stats.valor_total DESC;
```

---

### 📁 Exercício 5: Subquery no SELECT

```sql
-- Mostrar total de pedidos ao lado de cada usuário
SELECT 
    nome,
    (SELECT COUNT(*) FROM pedidos WHERE usuario_id = u.id) AS total_pedidos,
    (SELECT SUM(preco) FROM pedidos WHERE usuario_id = u.id) AS valor_total
FROM usuarios u;
```
