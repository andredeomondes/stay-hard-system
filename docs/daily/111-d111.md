## Setup com Vite

```bash
npm create vite@latest stay-hard-react -- --template react
cd stay-hard-react
npm install
npm run dev
```

## Componente Básico

```jsx
// src/App.jsx
function App() {
    return (
        <div className="app">
            <h1>🔥 Stay Hard</h1>
            <p>Gerenciador de Tarefas</p>
        </div>
    );
}

export default App;
```

## JSX é JavaScript + HTML

```jsx
// JSX
const elemento = <h1 className="titulo">Olá, {nome}!</h1>;

// Equivalente JavaScript
const elemento = React.createElement('h1', { className: 'titulo' }, `Olá, ${nome}!`);

// Expressões JavaScript
const bemVindo = <p>Bem-vindo, {user.name}!</p>;

// Condicional
const status = tarefa.concluida ? '✅' : '⬜';

// Array.map para listas
const lista = tarefas.map(t => (
    <li key={t.id}>{t.nome}</li>
));
```

## CSS Modules

```jsx
// App.module.css
.titulo {
    color: #6366f1;
}

// App.jsx
import styles from './App.module.css';

function App() {
    return <h1 className={styles.titulo}>Olá</h1>;
}
```
