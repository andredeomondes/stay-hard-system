# Dia 148 — IA: Agentes

## Conceito do Dia

**Agentes** são LLMs que usam tools de forma autônoma para completar tarefas complexas de múltiplos passos.

---

## Onde Estudar

- [Artigo: AI Agents](https://docs.spring.io/spring-ai/reference/1.0/concepts/agents.html)

---

## Conceito

```
Agente = LLM + Tools + Loop de Execução

1. LLM recebe task
2. LLM decide qual tool usar
3. Tool executa
4. LLM recebe resultado
5. LLM decide próximo passo
6. Repete até completar
```

## Tool simples

```java
public interface TarefaAgentTool {
    String getName();
    String getDescription();
    Object execute(Object input);
}

@Component
public class BuscarTarefasTool implements TarefaAgentTool {
    
    @Override
    public String getName() { return "buscar_tarefas"; }
    
    @Override
    public String getDescription() {
        return "Busca tarefas por categoria ou status. " +
               "Input: {categoria: 'ESTUDO', concluida: false}";
    }
    
    @Override
    public Object execute(Object input) {
        // parse input e executar

    }
}
```

## Agente com Loop

```java
@Service
@RequiredArgsConstructor
public class TarefaAgenteService {
    
    private final ChatClient chatClient;
    private final List<TarefaAgentTool> tools;
    
    public String executarTarefa(String instrucao) {
        String contexto = construirContextoTools();
        String prompt = """
            Você é um agente de produtividade. Use as tools disponíveis.
            
            Tools disponíveis:
            %s
            
            Tarefa: %s
            
            Responda no formato:
            TOOL: nome_da_tool
            INPUT: {"param": "valor"}
            
            Quando terminar, responda:
            RESPONSE: sua_resposta_final
            """.formatted(contexto, instrucao);
        
        String resposta = chatClient.call(prompt);
        
        // Executar tool chamada
        String toolName = extrairToolName(resposta);
        String toolInput = extrairToolInput(resposta);
        
        Object resultado = executarTool(toolName, toolInput);
        
        // Se precisar de mais steps, continuar loop
        if (precisaContinuar(resultado)) {
            return executarTarefa(instrucao + "\n\nResultado anterior: " + resultado);
        }
        
        return resultado.toString();
    }
}
```

## Exemplo de Uso

```
Tarefa: Me liste as 3 tarefas mais importantes de hoje

Loop 1:
→ LLM decide: TOOL: buscar_tarefas, INPUT: {status: "pendente"}
← Resultado: 5 tarefas pendentes

Loop 2:
→ LLM decide: TOOL: priorizar, INPUT: {tarefas: [...], criterio: "prioridade"}
← Resultado: [Alta, Alta, Media, Baixa, Baixa]

RESPONSE: Suas 3 tarefas mais importantes são:
1. Revisar documentação
2. Preparar apresentação
3. Ligar para cliente
```
