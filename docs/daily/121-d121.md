## Conceitos

| Termo | Significado |
|-------|-------------|
| Image | Template imutável (como classe) |
| Container | Instância em execução (como objeto) |
| Dockerfile | Receita para criar imagem |
| Registry | Repositório de imagens (Docker Hub) |

## Dockerfile

```dockerfile
# Dockerfile para Java app
FROM eclipse-temurin:21-jdk-alpine

WORKDIR /app

COPY pom.xml .
COPY src ./src

RUN apk add --no-cache maven && \
    mvn clean package -DskipTests && \
    rm -rf ~/.m2

FROM eclipse-temurin:21-jre-alpine

WORKDIR /app

COPY --from=0 /app/target/*.jar app.jar

EXPOSE 8080

ENTRYPOINT ["java", "-jar", "app.jar"]
```

## Comandos básicos

```bash
# Build
docker build -t stay-hard-api .

# Run
docker run -p 8080:8080 stay-hard-api

# Run em background
docker run -d -p 8080:8080 --name stayhard-api stay-hard-api

# Listar containers
docker ps
docker ps -a

# Parar container
docker stop stayhard-api

# Remover container
docker rm stayhard-api

# Ver logs
docker logs -f stayhard-api
```
