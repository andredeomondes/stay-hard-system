## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**Map — Coleção de pares chave-valor**

### Definição em 1 frase:
Map armazena dados como pares (chave → valor), onde cada chave é única.

### Exemplo do dia-a-dia:
Dicionário: "cachorro" → "dog", "gato" → "cat". Cada palavra (chave) tem um significado (valor).

### 📁 Exercício 1: HashMap Básico

```java
package com.estudo;

import java.util.*;

public class MapDemo {
    public static void main(String[] args) {
        // Criar mapa
        Map<String, Integer> idadePorNome = new HashMap<>();
        
        // Adicionar pares
        idadePorNome.put("André", 25);
        idadePorNome.put("Maria", 30);
        idadePorNome.put("João", 28);
        
        // Buscar por chave
        int idade = idadePorNome.get("André"); // 25
        Integer idadeNula = idadePorNome.get("Pedro"); // null
        
        // Atualizar valor
        idadePorNome.put("André", 26); // Sobrescreve valor antigo
        
        // Remover
        idadePorNome.remove("João");
        
        // Tamanho
        int tamanho = idadePorNome.size(); // 2
        
        // Verificar
        boolean existe = idadePorNome.containsKey("André");
        boolean temIdade25 = idadePorNome.containsValue(25);
        
        System.out.println(idadePorNome);
    }
}
```

### 📁 Exercício 2: Iterar sobre Map

```java
Map<String, Integer> mapa = Map.of(
    "André", 25,
    "Maria", 30,
    "João", 28
);

// 1. Iterar chaves
System.out.println("Chaves:");
for (String chave : mapa.keySet()) {
    System.out.println(chave);
}

// 2. Iterar valores
System.out.println("Valores:");
for (Integer valor : mapa.values()) {
    System.out.println(valor);
}

// 3. Iterar chaves e valores
System.out.println("Pares:");
for (Map.Entry<String, Integer> entry : mapa.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}

// 4. Lambda (mais moderno)
mapa.forEach((chave, valor) -> 
    System.out.println(chave + " = " + valor)
);
```

**Checklist:**
- [ ] keySet()
- [ ] values()
- [ ] entrySet()
- [ ] forEach com lambda

### 📁 Exercício 3: Map de Objetos

```java
// Map com objetos
Map<String, Usuario> usuarios = new HashMap<>();

usuarios.put("andre", new Usuario("André", 25));
usuarios.put("maria", new Usuario("Maria", 30));

// Buscar
Usuario u = usuarios.get("andre");
System.out.println(u.getNome()); // André

// Null safety
Usuario不存在 = usuarios.get("pedro"); // null
Usuario或Else = usuarios.getOrDefault("pedro", new Usuario("Default", 0));
```

**Checklist:**
- [ ] Map de objetos

### 📁 Exercício 4: Contador com Map

```java
// Contar frequência de palavras
String texto = "java java python ruby java python java";
String[] palavras = texto.split(" ");

Map<String, Integer> contagem = new HashMap<>();

for (String palavra : palavras) {
    contagem.put(palavra, contagem.getOrDefault(palavra, 0) + 1);
}

System.out.println(contagem);
// {java=3, python=2, ruby=1}

// Ou usando merge
Map<String, Integer> contagem2 = new HashMap<>();
for (String palavra : palavras) {
    contagem2.merge(palavra, 1, Integer::sum);
}
```

**Checklist:**
- [ ] getOrDefault para contar
- [ ] merge para contar
