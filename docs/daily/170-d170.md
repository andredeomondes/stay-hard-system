## 🚀 Quick Start

### Prerequisites

- Java 21+
- Node.js 20+
- Docker

### Run

```bash
# Clone
git clone https://github.com/seuuser/stayhard-api.git
cd stayhard-api

# Docker (recomendado)
docker-compose up -d

# Ou local
./mvnw spring-boot:run
```

## 🔧 Configuração

```yaml
# application.yml
spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/stayhard
  jpa:
    hibernate:
      ddl-auto: validate
```

## 📝 API Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| POST | /api/auth/register | Registrar |
| POST | /api/auth/login | Login |
| GET | /api/tarefas | Listar tarefas |
| POST | /api/tarefas | Criar tarefa |
| PUT | /api/tarefas/{id} | Atualizar |
| DELETE | /api/tarefas/{id} | Deletar |
