## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: curl GET

```bash
# GET simples
curl https://jsonplaceholder.typicode.com/users

# GET com pretty print (com jq)
curl -s https://jsonplaceholder.typicode.com/users | jq

# GET com query params
curl "https://jsonplaceholder.typicode.com/posts?userId=1"

# GET com headers
curl -H "Accept: application/json" \
     https://jsonplaceholder.typicode.com/users/1
```

### 📁 Exercício 2: curl POST

```bash
# POST criar usuário
curl -X POST https://jsonplaceholder.typicode.com/users \
  -H "Content-Type: application/json" \
  -d '{
    "name": "André de Omondes",
    "username": "andredev",
    "email": "andre@email.com",
    "address": {
      "street": "Rua Java",
      "city": "São Paulo"
    }
  }'
```

**Resposta esperada:**
```json
{
  "name": "André de Omondes",
  "username": "andredev",
  "email": "andre@email.com",
  "id": 11
}
```

**Checklist:**
- [ ] POST criar recurso
- [ ] Ver status 201
- [ ] Ver ID retornado
- [ ] Criar mais 2 recursos

### 📁 Exercício 3: API Testing Script

**Objetivo:** Criar script para testar API automaticamente

```bash
# Salvar como test-api.sh

#!/bin/bash

echo "=== Testando API ==="

# GET all users
echo -e "\n1. GET /users"
curl -s https://jsonplaceholder.typicode.com/users | jq '.[0:2]'

# GET single user
echo -e "\n2. GET /users/1"
curl -s https://jsonplaceholder.typicode.com/users/1 | jq '{name, email}'

# POST new user
echo -e "\n3. POST /users"
curl -s -X POST https://jsonplaceholder.typicode.com/users \
  -H "Content-Type: application/json" \
  -d '{"name": "Test User", "username": "test", "email": "test@email.com"}' | jq '{id, name}'

echo -e "\n=== Testes concluídos ==="
```

**Checklist:**
- [ ] Criar `test-api.sh`
- [ ] chmod +x test-api.sh
- [ ] Executar
- [ ] Ver resultados

## 🔧 COMANDOS DO DIA

