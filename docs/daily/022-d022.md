## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: @DisplayName

```java
class UsuarioTest {
    
    @Test
    @DisplayName("Deve criar usuário com nome válido")
    void deveCriarUsuarioComNomeValido() {
        Usuario user = new Usuario("André");
        assertEquals("André", user.getNome());
    }
    
    @Test
    @DisplayName("Deve lançar exceção quando nome for nulo")
    void deveLancarExcecaoQuandoNomeForNulo() {
        assertThrows(IllegalArgumentException.class, 
            () -> new Usuario(null)
        );
    }
}
```

---

### 📁 Exercício 2: assertAll (Agrupar assertions)

```java
@Test
@DisplayName("Deve criar usuário com todos os campos")
void deveCriarUsuarioCompleto() {
    Usuario user = new Usuario("André", 25, "andre@email.com");
    
    // Agrupar assertions - todas são executadas
    assertAll(
        () -> assertEquals("André", user.getNome()),
        () -> assertEquals(25, user.getIdade()),
        () -> assertEquals("andre@email.com", user.getEmail()),
        () -> assertTrue(user.isAtivo())
    );
}
```

---

### 📁 Exercício 3: assertThrows

```java
@Test
@DisplayName("Deve lançar exceção para divisão por zero")
void deveLancarExcecaoDivisaoPorZero() {
    Calculadora calc = new Calculadora();
    
    ArithmeticException ex = assertThrows(
        ArithmeticException.class,
        () -> calc.dividir(10, 0)
    );
    
    // Verificar mensagem da exceção
    assertEquals("Divisão por zero", ex.getMessage());
}
```

---

### 📁 Exercício 4: @ParameterizedTest

```java
@ParameterizedTest
@DisplayName("Soma deve retornar resultado correto")
@CsvSource({
    "1, 2, 3",
    "0, 0, 0",
    "-1, 1, 0",
    "10, 20, 30"
})
void testSomaParametrizado(int a, int b, int esperado) {
    Calculadora calc = new Calculadora();
    assertEquals(esperado, calc.somar(a, b));
}
```

**Dependencies:**
```xml

</dependency>
```
