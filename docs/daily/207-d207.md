# Dia 207 — Mentoria: Ensinar e Aprender

## Conceito do Dia

Mentoria beneficia ambos: mentor e mentored.

---

## Para Mentores

```markdown
## Como ser um bom mentor

1. **Ouvir mais, falar menos**
   - Perguntas > respostas
   - "O que você já tentou?"

2. **Compartilhar experiências**
   - Casos reais (sucessos e falhas)
   - Contexto de decisões

3. **Dar feedback construtivo**
   - Específico
   - Açãoável
   - Com exemplos

4. **Mentorar para independência**
   - Ensinar a pescar
   - Não fazer pelo outro

## Estrutura de Sessão

```
1. Check-in (5 min)
   - Como foi a semana?
   - Como está se sentindo?

2. Topic (40 min)
   - Desafio atual
   - Brainstorm juntos
   - Próximos passos

3. Action items (5 min)
   - 1-3 ações concretas
   - Data para próxima sessão
```

## Para Mentored (Aprendizes)

```markdown
## Como aproveitar mentoria

1. **Ser proativo**
   - Preparar agenda antes
   - Ler materiais sugeridos
   - Implementar conselhos

2. **Pedir feedback específico**
   ❌ "O que acha de mim?"
   ✅ "Como posso melhorar minha comunicação técnica?"

3. **Aceitar críticas**
   - Feedback é presente
   - Não se defender
   - Agradecer

4. **Documentar aprendizados**
   - Notas de cada sessão
   - Progress tracker
   - Aplicar conceitos
```
