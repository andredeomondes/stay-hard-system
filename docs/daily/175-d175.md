# Dia 175 — Entrevista: Behavioral Questions

## Conceito do Dia

Perguntas comportamentais testam soft skills e fit cultural.

---

## Framework STAR

```
S - Situation: Contexto
T - Task: Sua responsabilidade
A - Action: O que você fez
R - Result: Resultado
```

## Exemplos

### "Conte sobre um desafio técnico"

> **S:** No meu último projeto, o sistema estava muito lento com 10k usuários.
> 
> **T:** Eu precisava identificar e resolver o gargalo de performance.
> 
> **A:** Usei profiling com VisualVM, identifiquei queries N+1 no JPA, adicionei índices no banco, implementei caching com Redis.
> 
> **R:** Reduzi tempo de resposta de 3s para 200ms. O time adotou práticas de performance.

### "Como você lida com prazos apertados?"

> Planejo tarefas por prioridade, foco no essencial, e comunico riscos antecipadamente. Se necessário, proponho reduzir escopo mantendo valor.

### "Descreva um conflito com colega"

> Tive divergência sobre arquitetura. Escutei os argumentos, propus POC para validar ambas abordagens, data mostrou a melhor solução.

## Qualidades Buscadas

| Qualidade | Como Demonstrar |
|-----------|------------------|
| Comunicação | Explique claramente |
| Trabalho em time | Mencione colaboração |
| Resolução de problemas | Dê exemplos concretos |
| Crescimento | Mostre curiosidade |
| Ownership | Fale de iniciativas próprias |
