# Dia 55 — JSP: Java Server Pages

## Conceito do Dia

**JSP** permite misturar HTML com código Java. É uma view technology - a parte "visual" do Java EE antigo. Hoje é menos usado (Thymeleaf, React etc), mas é importante entender.

## Servlet enviando dados

```java
@WebServlet("/tarefas")
public class ListaTarefasServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) 
            throws ServletException, IOException {
        
        TarefaService service = new TarefaService();
        List<Tarefa> tarefas = service.listarTodas();
        
        // Passar dados para a JSP
        req.setAttribute("tarefas", tarefas);
        req.getRequestDispatcher("/WEB-INF/views/tarefas.jsp").forward(req, resp);
    }
}
```

## JSP mostrando dados

```jsp
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%@ taglib prefix="c" uri="jakarta.tags.core" %>

<!DOCTYPE html>
<html>
<head>
    <title>Minhas Tarefas</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>📋 Minhas Tarefas</h1>
    
    <c:forEach var="tarefa" items="${tarefas}">
        <div class="tarefa ${tarefa.concluida ? 'concluida' : ''}">
            <h3>${tarefa.nome}</h3>
            <p>Prioridade: ${tarefa.prioridade}</p>
            <p>Categoria: ${tarefa.categoria}</p>
        </div>
    </c:forEach>
</body>
</html>
```

## JSTL para formatação

```jsp
<%@ taglib prefix="fmt" uri="jakarta.tags.fmt" %>

<fmt:formatDate value="${tarefa.criadoEm}" pattern="dd/MM/yyyy HH:mm"/>
```
