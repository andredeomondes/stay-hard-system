### 📁 Exercício 1: Adicionar Dependência

**1. Buscar no Maven Repository:**
```
1. Acesse: mvnrepository.com
2. Pesquise: "jackson"
3. Clique em jackson-databind
4. Copie a tag <dependency>
```

**2. Adicionar no pom.xml:**
```xml
<dependencies>
    <!-- JUnit (já existente) -->
    <dependency>
        <groupId>org.junit.jupiter</groupId>
        <artifactId>junit-jupiter</artifactId>
        <version>5.10.0</version>
        <scope>test</scope>
    </dependency>
    
    <!-- Adicionar: Jackson JSON -->
    <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-databind</artifactId>
        <version>2.15.3</version>
    </dependency>
</dependencies>
```

**3. Compilar (Maven baixa automaticamente):**
```bash
mvn compile
# Verá: Downloading from central: jackson-databind-2.15.3.jar
```

**Checklist:**
- [ ] Adicionou Jackson
- [ ] `mvn compile` baixou automaticamente
- [ ] Verifica em `.m2/repository`

### 📁 Exercício 2: Scopes (Escopos)

**Scopes controlam quando a dependência está disponível:**

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.0</version>
    <scope>test</scope>  <!-- Só em src/test/java -->
</dependency>

<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version>1.18.30</version>
    <scope>provided</scope>  <!-- Fornecido pelo container -->
</dependency>

<dependency>
    <groupId>com.fasterxml.jackson.core</groupId>
    <artifactId>jackson-databind</artifactId>
    <version>2.15.3</version>
    <!-- Sem scope = compile (padrão) -->
</dependency>
```

| Scope | Disponível em | No JAR final? |
|-------|---------------|----------------|
| `compile` | main + test | ✅ Sim |
| `test` | só test | ❌ Não |
| `provided` | main + test | ❌ Não |
| `runtime` | só runtime | ✅ Sim |

**Checklist:**
- [ ] Entender cada scope
- [ ] Adicionar lombok (provided)
- [ ] Ver diferença

### 📁 Exercício 3: Usar Dependência

**Usar Jackson para JSON:**
```java
package com.estudo;

import com.fasterxml.jackson.databind.ObjectMapper;

public class JsonDemo {
    public static void main(String[] args) throws Exception {
        // Criar objeto
        Usuario usuario = new Usuario("André", 25);
        
        // Converter para JSON (serializar)
        ObjectMapper mapper = new ObjectMapper();
        String json = mapper.writeValueAsString(usuario);
        System.out.println(json);
        // {"name":"André","age":25}
        
        // Converter de JSON para objeto (desserializar)
        String jsonInput = "{\"name\":\"Maria\",\"age\":30}";
        Usuario usuario2 = mapper.readValue(jsonInput, Usuario.class);
        System.out.println(usuario2.getName()); // Maria
    }
}

class Usuario {
    private String name;
    private int age;
    
    public Usuario() {} // Jackson precisa
    
    public Usuario(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public String getName() { return name; }
    public int getAge() { return age; }
}
```

**Checklist:**
- [ ] Criar JsonDemo.java
- [ ] Usar ObjectMapper
- [ ] Serializar objeto para JSON
- [ ] Desserializar JSON para objeto
- [ ] `mvn compile && java com.estudo.JsonDemo`
