## Side Project Checklist

```markdown
## Validação de Ideia

1. **Problema real?**
   - Você mesmo tem esse problema?
   - Outros também têm?

2. **Size of market?**
   - TAM: Total addressable
   - SAM: Serviceable
   - SOM: Obtainable

3. **Competitive landscape?**
   - Quais soluções existem?
   - Como seu diferencial?

4. **Revenue model?**
   - Quanto cobrar?
   - CAC vs LTV?
```

## Roadmap de Monetização

```markdown
## Stay Hard Monetization (exemplo)

### Fase 1: Freemium
- Free: 50 tarefas, 1 projeto
- Pro ($9/mês): Ilimitado + IA

### Fase 2: Team ($29/usuário)
- Multi-usuário
- Admin dashboard
- Audit log

### Fase 3: Marketplace
- Templates de projetos
- Integrações (Slack, Notion)

### Fase 4: Enterprise
- SSO/SAML
- SLA 99.9%
- Dedicated support
```
