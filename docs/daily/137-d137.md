## Producer

```java
@Service
@RequiredArgsConstructor
public class TarefaEventPublisher {
    
    private final RabbitTemplate rabbitTemplate;
    
    public void publicarTarefaCriada(Tarefa tarefa) {
        TarefaMessage message = new TarefaMessage(
            tarefa.getId(),
            tarefa.getNome(),
            tarefa.getUsuarioId()
        );
        
        rabbitTemplate.convertAndSend(
            "tarefas.exchange",
            "tarefa.criada",
            message
        );
    }
}
```

## Config de Exchanges e Queues

```java
@Configuration
public class RabbitMQConfig {
    
    public static final String EXCHANGE = "tarefas.exchange";
    public static final String QUEUE = "tarefa.criada.queue";
    public static final String ROUTING_KEY = "tarefa.criada";
    
    @Bean
    Queue queue() {
        return QueueBuilder.durable(QUEUE).build();
    }
    
    @Bean
    TopicExchange exchange() {
        return new TopicExchange(EXCHANGE);
    }
    
    @Bean
    Binding binding(Queue queue, TopicExchange exchange) {
        return BindingBuilder.bind(queue).to(exchange).with(ROUTING_KEY);
    }
}
```
