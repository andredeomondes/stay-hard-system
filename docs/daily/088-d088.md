# Dia 88 — REST API: OpenAPI/Swagger

## Conceito do Dia

**OpenAPI (Swagger)** documenta sua API automaticamente. Outros devs ou ferramentas podem entender como usar sua API.

## Dependência

```xml
<dependency>
    <groupId>org.springdoc</groupId>
    <artifactId>springdoc-openapi-starter-webmvc-ui</artifactId>
    <version>2.3.0</version>
</dependency>
```

## Configuração

```yaml
springdoc:
  api-docs:
    path: /api-docs
  swagger-ui:
    path: /swagger-ui.html
    operations-sorter: method
    tags-sorter: alpha
  show-actuator-endpoints: true
```

## Anotações

```java

@RequestMapping("/api/tarefas")
@Tag(name = "Tarefas", description = "Gerenciamento de tarefas")
public class TarefaController {
    
    @Operation(summary = "Listar todas as tarefas", description = "Retorna lista paginada")
    @ApiResponses({
        @ApiResponse(responseCode = "200", description = "Lista de tarefas"),
        @ApiResponse(responseCode = "401", description = "Não autorizado")
    })
    @GetMapping
    public ResponseEntity<Page<TarefaDTO>> listar(
            @Parameter(description = "Número da página") @RequestParam(defaultValue = "0") int pagina) {
        // ...
    }
    
    @Operation(summary = "Criar tarefa", description = "Cria nova tarefa")
    @ApiResponse(responseCode = "201", description = "Tarefa criada")
    @PostMapping
    public ResponseEntity<TarefaDTO> criar(
            @io.swagger.v3.oas.annotations.parameters.RequestBody(
                description = "Dados da tarefa", required = true)
            @Valid @RequestBody CriarTarefaRequest request) {
        // ...
    }
}
```
