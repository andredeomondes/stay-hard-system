## Métricas Customizadas

```java
@Service
public class TarefaMetrics {
    
    private final MeterRegistry registry;
    
    public TarefaMetrics(MeterRegistry registry) {
        this.registry = registry;
        
        // Contador
        Counter.builder("tarefa.criada.total")
            .description("Total de tarefas criadas")
            .tag("categoria", "geral")
            .register(registry);
        
        // Gauge
        Gauge.builder("tarefa.pendentes", repository, TarefaRepository::countPendentes)
            .description("Tarefas pendentes")
            .register(registry);
        
        // Timer
        Timer.builder("tarefa.criacao.time")
            .description("Tempo de criação")
            .register(registry);
    }
    
    public void contarTarefaCriada() {
        registry.counter("tarefa.criada.total").increment();
    }
}
```

## Controller com Timer

```java
@RestController
public class TarefaController {
    
    private final MeterRegistry registry;
    
    @GetMapping("/api/tarefas")
    public List<Tarefa> listar() {
        Timer.Sample sample = Timer.start(registry);
        
        List<Tarefa> tarefas = service.listar();
        
        sample.stop(Timer.builder("tarefa.listar.time")
            .tag("status", "success")
            .register(registry));
        
        return tarefas;
    }
}
```

## Prometheus Query

```promql
# Requests por segundo
rate(http_server_requests_seconds_count{uri="/api/tarefas"}[5m])

# Tempo médio de resposta
rate(http_server_requests_seconds_sum{uri="/api/tarefas"}[5m]) /
rate(http_server_requests_seconds_count{uri="/api/tarefas"}[5m])

# Tarefas criadas por minuto
increase(tarefa_criada_total[1m])

# Uso de memória
jvm_memory_used_bytes{area="heap"}
```
