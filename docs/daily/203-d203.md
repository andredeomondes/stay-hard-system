## Checklist de Oferta

```markdown
## Ao receber oferta

### Salário
- [ ] Salário base
- [ ] Bônus (anual %)
- [ ] Equity (stock options)

### Benefícios
- [ ] Plano de saúde (coparticipação?)
- [ ] VR/VA (valor diário)
- [ ] Transporte/Estacionamento
- [ ] Gym/Pilates
- [ ] Auxílio home office
- [ ] celular corporativo

### Desenvolvimento
- [ ] Treinamentos/certificações
- [ ] Books/conferências
- [ ] Plano de carreira

### Trabalho
- [ ] Horário flexível
- [ ] Remote/híbrido
- [ ] Feriados e férias (28 dias?)
```

## Template de Resposta

```markdown
## Email de Contra-proposta

Prezado [Nome],

Agradeço pela oferta e estou muito animado com a oportunidade.

Após analisar a proposta, gostaria de discutir alguns pontos:

1. **Salário:** R$ X é abaixo do mercado para minha experiência. 
   Com base em minha pesquisa, o faixa é R$ Y-Z. Gostaria de 
   propor R$ W.

2. **Férias:** Actualmente tenho X dias. Gostaria de solicitar 
   aumento para Y dias.

3. **Benefícios:** Gostaria de discutir [benefício específico].

Estou comprometido com esta posição e acredito que estes ajustes 
tornarão a oferta mais alinhada com minhas expectativas.

Fico à disposição para discutir.

Atenciosamente,
[Seu nome]
```
