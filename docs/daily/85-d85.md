## URL Versioning (mais simples)

```
GET /api/v1/tarefas
GET /api/v2/tarefas
```

```java
@RestController
@RequestMapping("/api/v2/tarefas")
public class TarefaControllerV2 {
    
    @GetMapping
    public ResponseEntity<List<TarefaDTO>> listar() {
        // Retorna DTO em vez de entidade
        return ResponseEntity.ok(service.listar().stream()
            .map(TarefaDTO::new)
            .toList());
    }
}
```

## Header Versioning (mais RESTful)

```
GET /api/tarefas
Accept: application/vnd.stayhard.v2+json
```

```java
@GetMapping(produces = "application/vnd.stayhard.v2+json")
public ResponseEntity<List<TarefaDTO>> listarV2() {
    return ResponseEntity.ok(service.listar().stream()
        .map(TarefaDTO::new)
        .toList());
}
```

## Diferenças entre V1 e V2

| Aspecto | V1 | V2 |
|---------|----|----|
| Retorno | Entity | DTO |
| Campos | Todos | Filtrados |
| Formato data | ISO | dd/MM/yyyy |

---

## Estratégias

1. **URL** (simples, mais usado): `/v1/`, `/v2/`
2. **Header**: `API-Version: 2`
3. **Query**: `?version=2`
4. **Content-Type**: `application/vnd.api.v2+json`
