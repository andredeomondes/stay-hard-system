## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**Headers e Query Parameters**

### Definição em 1 frase:
Headers = metadados da requisição (como quem está pedindo)
Query Params = filtros na URL (após ?)

### Exemplo do dia-a-dia:
```
https://netflix.com/browse?genre=action&year=2023
                     │__________________│
                              │
                              └── Query Params (filtros)

Headers:
Authorization: Bearer token123
Content-Type: application/json
User-Agent: Mozilla/5.0
```

## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: Headers de Requisição

```bash
# Header único
curl -H "Content-Type: application/json" \
     https://api.exemplo.com/data

# Múltiplos headers
curl -H "Content-Type: application/json" \
     -H "Authorization: Bearer token123" \
     -H "Accept: application/json" \
     https://api.exemplo.com/data

# Header específico
curl -H "X-API-Key: minha-chave-secreta" \
     https://api.exemplo.com/data

# Header User-Agent
curl -H "User-Agent: MeuApp/1.0" \
     https://api.exemplo.com/data
```

**Checklist:**
- [ ] Content-Type header
- [ ] Authorization header
- [ ] Accept header
- [ ] X-API-Key header

---

### 📁 Exercício 2: Headers de Resposta

```bash
# Ver todos headers da resposta
curl -i https://jsonplaceholder.typicode.com/users/1

# Output esperado:
# HTTP/2 200
# content-type: application/json
# date: Thu, 29 Mar 2024...
# ...

# Só headers (HEAD request)
curl -I https://google.com
```

**Headers comuns de resposta:**
| Header | Significado |
|--------|------------|
| `Content-Type` | Tipo do conteúdo (json, html) |
| `Content-Length` | Tamanho em bytes |
| `Set-Cookie` | Cookie para salvar |
| `Cache-Control` | Como cachear |
| `Location` | Onde foi criado (201) |

---

### 📁 Exercício 3: Query Parameters

```bash
# Filtro simples
curl "https://jsonplaceholder.typicode.com/posts?userId=1"

# Múltiplos filtros
curl "https://jsonplaceholder.typicode.com/posts?userId=1&completed=true"

# URL encoding (espaços)
curl "https://api.com/search?q=java%20spring"

# Pipe com jq para filtrar
curl -s "https://jsonplaceholder.typicode.com/posts?userId=1" | jq '.[].title'
```

**Checklist:**
- [ ] Query params simples
- [ ] Múltiplos query params
- [ ] jq para filtrar resultados
- [ ] URL encoding

---

### 📁 Exercício 4: Combinar Headers + Query Params

```bash
# Headers + Query Params
curl -X GET "https://api.exemplo.com/users?city=São Paulo" \
  -H "Authorization: Bearer token123" \
  -H "Accept: application/json"
```

**Checklist:**
- [ ] Combinar headers com query params
- [ ] Verificar que headers vão antes da URL
- [ ] Verificar que query params vão depois de ?

## 🔧 COMANDOS DO DIA

```bash
# Headers na requisição
curl -H "Header: valor" https://api.com/data

# Ver headers da resposta
curl -i https://api.com/data

# Query params
curl "https://api.com/data?param1=valor1&param2=valor2"

# Combinar
curl -H "Authorization: Bearer token" \
     "https://api.com/data?filter=ativo" \
     -v
```
