## Stack Tecnológico

| Camada | Tecnologia |
|--------|------------|
| Gateway | Spring Cloud Gateway |
| Service Discovery | Eureka |
| API | Spring Boot 3 |
| Database | PostgreSQL |
| Cache | Redis |
| Messaging | Kafka |
| Tracing | OpenTelemetry + Jaeger |
| Metrics | Prometheus + Grafana |
| Container | Docker + Kubernetes |
| CI/CD | GitHub Actions |
