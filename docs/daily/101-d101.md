## Estrutura básica

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Stay Hard - Gerenciador de Tarefas</title>
</head>
<body>
    <header>
        <h1>Stay Hard</h1>
        <nav>
            <a href="/">Home</a>
            <a href="/tarefas">Tarefas</a>
        </nav>
    </header>
    
    <main>
        <section id="lista-tarefas">
            <h2>Minhas Tarefas</h2>
        </section>
    </main>
    
    <footer>
        <p>&copy; 2026 Stay Hard System</p>
    </footer>
</body>
</html>
```

## Tags importantes

| Tag | Uso |
|-----|-----|
| `<h1>` - `<h6>` | Títulos |
| `<p>` | Parágrafo |
| `<a>` | Link |
| `<ul>`, `<ol>` | Listas |
| `<li>` | Item de lista |
| `<div>` | Container genérico (block) |
| `<span>` | Container genérico (inline) |
| `<button>` | Botão |
| `<input>` | Campo de entrada |
| `<form>` | Formulário |

## Tags semânticas

```html
<header>   <!-- Cabeçalho -->
<nav>      <!-- Navegação -->
<main>     <!-- Conteúdo principal -->
<section>  <!-- Seção -->
<article>  <!-- Artigo/conteúdo independente -->
<aside>    <!-- Conteúdo lateral -->
<footer>   <!-- Rodapé -->
```
