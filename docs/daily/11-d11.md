### 📁 Exercício 2: Criar Projeto Maven

```bash
# Criar com archetype (template)
mvn archetype:generate \
    -DgroupId=com.estudo \
    -DartifactId=meu-primeiro-projeto \
    -DarchetypeArtifactId=maven-archetype-quickstart \
    -DinteractiveMode=false

# Estrutura gerada:
# meu-primeiro-projeto/
# ├── pom.xml
# └── src/
#     ├── main/java/com/estudo/
#     │   └── App.java
#     └── test/java/com/estudo/
#         └── AppTest.java
```

**Checklist:**
- [ ] `mvn archetype:generate`
- [ ] Ver estrutura criada
- [ ] `mvn --version` funciona

### 📁 Exercício 3: Entender pom.xml

**Criar pom.xml manualmente:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 
         http://maven.apache.org/xsd/maven-4.0.0.xsd">
    
    <!-- Versão do POM (sempre 4.0.0) -->
    <modelVersion>4.0.0</modelVersion>
    
    <!-- Identificação do projeto -->
    <groupId>com.estudo</groupId>
    <artifactId>meu-projeto</artifactId>
    <version>1.0.0</version>
    <packaging>jar</packaging>
    
    <!-- Nome e descrição -->
    <name>Meu Projeto</name>
    <description>Primeiro projeto Maven</description>
    
    <!-- Propriedades -->
    <properties>
        <maven.compiler.source>21</maven.compiler.source>
        <maven.compiler.target>21</maven.compiler.target>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>
    
    <!-- Dependências -->
    <dependencies>
        <!-- JUnit para testes -->
        <dependency>
            <groupId>org.junit.jupiter</groupId>
            <artifactId>junit-jupiter</artifactId>
            <version>5.10.0</version>
            <scope>test</scope>
        </dependency>
    </dependencies>
    
    <!-- Build -->
    <build>
        <plugins>
            <plugin>
                <groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-surefire-plugin</artifactId>
                <version>3.2.0</version>
            </plugin>
        </plugins>
    </build>
</project>
```

**Partes importantes:**
- [ ] `groupId` — domínio invertido (com.empresa)
- [ ] `artifactId` — nome do projeto
- [ ] `version` — versão
- [ ] `properties` — configurações
- [ ] `dependencies` — libs externas
- [ ] `build/plugins` — plugins de build

### 📁 Exercício 4: Estrutura de Pastas

```
meu-projeto/
├── pom.xml                     ← Configuração Maven
├── src/
│   ├── main/
│   │   ├── java/              ← Código fonte (.java)
│   │   └── resources/         ← Arquivos de config
│   └── test/
│       └── java/              ← Testes (.java)
└── target/                     ← Build output (gerado pelo Maven)
```

**Checklist:**
- [ ] Entender cada pasta
- [ ] `src/main/java` = código
- [ ] `src/test/java` = testes
- [ ] `target` = compilado

## 🔧 COMANDOS DO DIA

```bash
# Verificar Maven
mvn --version

# Criar projeto
mvn archetype:generate -DgroupId=com.estudo -DartifactId=projeto -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

# Compilar
mvn compile

# Rodar testes
mvn test

# Limpar + compilar + testar
mvn clean test

# Gerar JAR
mvn package

# Ver árvore de dependências
mvn dependency:tree

# Limpar tudo
mvn clean
```
