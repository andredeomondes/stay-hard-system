## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**HTTP — HyperText Transfer Protocol**

### Definição em 1 frase:
HTTP é o protocolo que navegadores usam para pedir páginas web e servidores usam para responder.

### Exemplo do dia-a-dia:
Você digita "google.com" → navegador envia REQUEST HTTP → servidor responde com HTML.

```java
package com.estudo;

import java.net.URI;
import java.net.http.*;

public class HttpClientDemo {
    public static void main(String[] args) throws Exception {
        // TODO 1: Criar HttpClient
        HttpClient client = HttpClient.newHttpClient();
        
        // TODO 2: Criar requisição GET
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://jsonplaceholder.typicode.com/users/1"))
            .GET() // Método GET
            .build();
        
        // TODO 3: Enviar e receber resposta
        HttpResponse<String> response = client.send(request, 
            HttpResponse.BodyHandlers.ofString()
        );
        
        // TODO 4: Exibir resultado
        System.out.println("Status: " + response.statusCode());
        System.out.println("Body: " + response.body());
    }
}
```

### 📁 Exercício 2: HTTP POST Request

**Objetivo:** Enviar dados via POST

**Código:**
```java
package com.estudo;

import java.net.URI;
import java.net.http.*;

public class HttpPostDemo {
    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();
        
        // TODO 1: Criar JSON body
        String jsonBody = """
            {
                "name": "André de Omondes",
                "username": "andredeomondes",
                "email": "andre@email.com"
            }
            """;
        
        // TODO 2: Criar requisição POST
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://jsonplaceholder.typicode.com/users"))
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(jsonBody))
            .build();
        
        // TODO 3: Enviar
        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString()
        );
        
        // TODO 4: Ver resultado
        System.out.println("Status: " + response.statusCode());
        System.out.println("Response: " + response.body());
    }
}
```

### 📁 Exercício 3: API Testing (similar ao curl)

**Objetivo:** Criar utilitário para testar APIs

**Código:**
```java
package com.estudo;

import java.net.URI;
import java.net.http.*;

public class ApiTester {
    private final HttpClient client = HttpClient.newHttpClient();
    
    public String get(String url) throws Exception {
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .GET()
            .build();
        
        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString()
        );
        
        return formatResponse(response);
    }
    
    public String post(String url, String body) throws Exception {
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(body))
            .build();
        
        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString()
        );
        
        return formatResponse(response);
    }
    
    private String formatResponse(HttpResponse<String> response) {
        return """
            Status: %d
            Body: %s
            """.formatted(response.statusCode(), response.body());
    }
}
```

- [ ] Implementar `ApiTester.java`

## 📝 RESUMO TEÓRICO

### Métodos HTTP:

```
GET    → Ler dados
POST   → Criar dados
PUT    → Substituir dados
PATCH  → Atualizar parcialmente
DELETE → Remover dados
```

### Status Codes:

```
2xx → Sucesso
200 OK
201 Created
204 No Content

4xx → Erro do Cliente
400 Bad Request
401 Unauthorized
404 Not Found

5xx → Erro do Servidor
500 Internal Server Error
```
