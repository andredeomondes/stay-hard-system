# Dia 52 — Servlets: Introdução

## Conceito do Dia

**Servlet** é um componente Java que processa requisições HTTP. É a base do Java EE/Jakarta EE. Antes do Spring Boot, todo web app Java usava Servlets.

## pom.xml

```xml
<dependency>
    <groupId>jakarta.servlet</groupId>
    <artifactId>jakarta.servlet-api</artifactId>
    <version>6.0.0</version>
    <scope>provided</scope>
</dependency>
```

---

## Código: MeuPrimeiroServlet

```java
import jakarta.servlet.http.HttpServlet;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;

public class OlaMundoServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) 
            throws ServletException, IOException {
        
        resp.setContentType("text/html;charset=UTF-8");
        resp.getWriter().write("""
            <html>
                <body>
                    <h1>Olá Mundo Servlet!</h1>
                    <p>Data: """ + LocalDate.now() + """</p>
                </body>
            </html>
        """);
    }
}
```

## web.xml (mapeamento)

```xml
<web-app>
    <servlet>
        <servlet-name>olaMundo</servlet-name>
        <servlet-class>com.example.OlaMundoServlet</servlet-class>
    </servlet>
    
    <servlet-mapping>
        <servlet-name>olaMundo</servlet-name>
        <url-pattern>/ola</url-pattern>
    </servlet-mapping>
</web-app>
```
