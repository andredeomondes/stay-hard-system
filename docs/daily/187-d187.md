# Dia 187 — gRPC: Introdução

## Conceito do Dia

**gRPC** é framework de RPC de alta performance. Usa Protobuf para serialização binária.

---

## Por que gRPC?

```
- 10x mais rápido que REST JSON
- Type-safe com .proto
- Bidirectional streaming
- Code generation automático
```

## Protocol Buffers

```protobuf
// tarefa.proto
syntax = "proto3";
option java_package = "br.com.stayhard.grpc";
option java_multiple_files = true;

service TarefaService {
    rpc BuscarTarefa(BuscarTarefaRequest) returns (Tarefa);
    rpc ListarTarefas(ListarTarefasRequest) returns (stream Tarefa);
    rpc CriarTarefa(stream CriarTarefaRequest) returns (stream Tarefa);
}

message Tarefa {
    int64 id = 1;
    string nome = 2;
    string categoria = 3;
    string prioridade = 4;
    bool concluida = 5;
}

message BuscarTarefaRequest {
    int64 id = 1;
}

message ListarTarefasRequest {
    string categoria = 1;
}
```

## Server

```java
public class TarefaGrpcService extends TarefaServiceGrpc.TarefaServiceImplBase {
    
    private final TarefaService tarefaService;
    
    @Override
    public void buscarTarefa(BuscarTarefaRequest request, 
                             StreamObserver<TarefaProto> observer) {
        Optional<Tarefa> tarefa = tarefaService.buscarPorId(request.getId());
        
        if (tarefa.isPresent()) {
            observer.onNext(toProto(tarefa.get()));
            observer.onCompleted();
        } else {
            observer.onError(Status.NOT_FOUND.asException());
        }
    }
    
    @Override
    public void listarTarefas(ListarTarefasRequest request,
                             StreamObserver<TarefaProto> observer) {
        List<Tarefa> tarefas = tarefaService.listar();
        
        for (Tarefa t : tarefas) {
            observer.onNext(toProto(t));
        }
        observer.onCompleted();
    }
}
```
