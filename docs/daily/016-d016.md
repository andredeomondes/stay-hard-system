### 📁 Exercício 1: ArrayList Básico

```java
package com.estudo;

import java.util.*;

public class ListDemo {
    public static void main(String[] args) {
        // Criar lista
        List<String> nomes = new ArrayList<>();
        
        // Adicionar elementos
        nomes.add("André");
        nomes.add("Maria");
        nomes.add("João");
        System.out.println(nomes); // [André, Maria, João]
        
        // Adicionar em posição específica
        nomes.add(1, "Pedro");
        System.out.println(nomes); // [André, Pedro, Maria, João]
        
        // Acessar por índice
        String primeiro = nomes.get(0);
        System.out.println(primeiro); // André
        
        // Modificar
        nomes.set(0, "André Updated");
        System.out.println(nomes); // [André Updated, Pedro, Maria, João]
        
        // Remover
        nomes.remove(0);        // Por índice
        nomes.remove("Maria"); // Por objeto
        
        // Tamanho
        int tamanho = nomes.size();
        System.out.println("Tamanho: " + tamanho);
        
        // Verificar
        boolean tem = nomes.contains("Pedro");
        boolean vazia = nomes.isEmpty();
        
        System.out.println(nomes);
    }
}
```

- [ ] size(), contains(), isEmpty()

### 📁 Exercício 2: Iterar sobre List

```java
List<Integer> numeros = List.of(1, 2, 3, 4, 5);

// 1. For-each
for (Integer num : numeros) {
    System.out.println(num);
}

// 2. Lambda forEach
numeros.forEach(num -> System.out.println(num));

// 3. Method reference
numeros.forEach(System.out::println);

// 4. For tradicional
for (int i = 0; i < numeros.size(); i++) {
    System.out.println(i + ": " + numeros.get(i));
}
```

**Checklist:**
- [ ] For-each
- [ ] Lambda forEach
- [ ] Method reference
- [ ] For tradicional

### 📁 Exercício 3: List de Objetos

```java
// List de usuários
List<Usuario> usuarios = new ArrayList<>();

usuarios.add(new Usuario("André", 25));
usuarios.add(new Usuario("Maria", 30));

// Buscar usuário
for (Usuario u : usuarios) {
    if (u.getNome().equals("André")) {
        System.out.println("Encontrado: " + u.getIdade());
    }
}

// Usando stream
usuarios.stream()
    .filter(u -> u.getNome().equals("André"))
    .findFirst()
    .ifPresent(u -> System.out.println("Idade: " + u.getIdade()));
```

**Checklist:**
- [ ] List de objetos
- [ ] Buscar na lista
- [ ] Usar stream para buscar
