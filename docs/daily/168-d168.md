# Dia 168 — Agile: User Stories e BDD

## Conceito do Dia

**User Stories** descrevem funcionalidades do ponto de vista do usuário. **BDD** testa comportamento esperado.

---

## Template de User Story

```
COMO [pessoa]
QUERO [ação]
PARA [benefício]
```

## Exemplos

```markdown
COMO usuário autenticado
QUERO criar uma tarefa
PARA organizar minhas atividades

Critérios de aceitação:
- [ ] Tarefa aparece na lista após criar
- [ ] Campos obrigatórios são validados
- [ ] Prioridade padrão é MÉDIA

---

COMO usuário
QUERO receber lembretes
PARA não esquecer prazos

Critérios de aceitação:
- [ ] Lembrete 1 dia antes
- [ ] Lembrete no dia
- [ ] Email contém nome da tarefa
```

## BDD com Cucumber

```java
// src/test/resources/tarefas.feature
Feature: Gerenciamento de Tarefas
  
  Scenario: Criar tarefa com sucesso
    Given usuário está autenticado
    When usuário preenche formulário com:
      | nome        | Estudar Spring |
      | categoria   | ESTUDO         |
      | prioridade  | ALTA           |
    And clica em "Criar"
    Then tarefa aparece na lista
    And mensagem de sucesso é exibida

  Scenario: Criar tarefa sem nome
    Given usuário está autenticado
    When usuário deixa nome vazio
    And clica em "Criar"
    Then mensagem de erro é exibida
    And tarefa não é criada
```

## Step Definitions

```java
@SpringBootTest
@CucumberContextConfiguration
public class TarefaSteps {
    
    @Autowired
    private TarefaService service;
    
    @Given("usuário está autenticado")
    public void usuarioAutenticado() {
        // Setup auth
    }
    
    @When("usuário preenche formulário com:")
    public void preencheFormulario(Map<String, String> dados) {
        this.dadosTarefa = dados;
    }
    
    @When("clica em {string}")
    public void clicaEm(String botao) {
        // Interact with UI or service
    }
    
    @Then("tarefa aparece na lista")
    public void tarefaApareceNaLista() {
        List<Tarefa> tarefas = service.listarTodas();
        assertTrue(tarefas.stream()
            .anyMatch(t -> t.getNome().equals(dadosTarefa.get("nome"))));
    }
}
```
