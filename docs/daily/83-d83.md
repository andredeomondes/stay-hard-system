# Dia 83 — REST API: Pagination e Sorting

## Conceito do Dia

Não retorne 1 milhão de registros de uma vez. Use **paginação** para dividir em páginas menores.

## Repository com Pageable

```java
public interface TarefaRepository extends JpaRepository<Tarefa, Long> {
    
    Page<Tarefa> findAll(Pageable pageable);
    
    Page<Tarefa> findByCategoria(Categoria categoria, Pageable pageable);
    
    Page<Tarefa> findByNomeContainingIgnoreCase(String termo, Pageable pageable);
}
```

## Controller com paginação

```java
@GetMapping
public ResponseEntity<Page<Tarefa>>> listar(
        @RequestParam(defaultValue = "0") int pagina,
        @RequestParam(defaultValue = "10") int tamanho,
        @RequestParam(defaultValue = "criadoEm") String ordenacao,
        @RequestParam(defaultValue = "desc") String direcao) {
    
    Sort.Direction direction = direcao.equalsIgnoreCase("asc") 
        ? Sort.Direction.ASC 
        : Sort.Direction.DESC;
    
    Pageable pageable = PageRequest.of(pagina, tamanho, Sort.by(direction, ordenacao));
    
    Page<Tarefa> tarefas = service.listarPaginadas(pageable);
    
    return ResponseEntity.ok(tarefas);
}
```
