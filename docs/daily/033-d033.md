## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**Herança — Reutilizar código entre classes**

### Definição em 1 frase:
Herança permite criar uma classe baseada em outra, herdando seus atributos e métodos.

### Exemplo:
```
Animal (classe pai)
├── Mamifero (herda de Animal)
│   ├── Cao (herda de Mamifero)
│   └── Gato (herda de Mamifero)
└── Ave (herda de Animal)
    ├── Papagaio (herda de Ave)
    └── Pinguim (herda de Ave)
```

## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: extends

```java
// Classe pai
public class Funcionario {
    protected String nome; // protected = subclasses acessam
    protected double salario;
    
    public Funcionario(String nome, double salario) {
        this.nome = nome;
        this.salario = salario;
    }
    
    public void trabalhar() {
        System.out.println(nome + " trabalhando...");
    }
    
    public double getSalario() {
        return salario;
    }
}

// Classe filha
public class Gerente extends Funcionario {
    private String departamento;
    
    public Gerente(String nome, double salario, String departamento) {
        super(nome, salario); // Chama construtor do pai
        this.departamento = departamento;
    }
    
    public void fazerReuniao() {
        System.out.println(nome + " fazendo reunião...");
    }
    
    // pode usar getSalario() do pai
    public double getBonus() {
        return getSalario() * 0.1;
    }
}
```

**Uso:**
```java
Gerente g = new Gerente("Maria", 10000, "TI");
g.trabalhar();         // do pai
g.fazerReuniao();      // do gerente
System.out.println(g.getBonus()); // 1000
```

### 📁 Exercício 2: @Override

```java
public class Vendedor extends Funcionario {
    private double comissao;
    
    public Vendedor(String nome, double salario, double comissao) {
        super(nome, salario);
        this.comissao = comissao;
    }
    
    @Override // Sobrescreve método do pai
    public void trabalhar() {
        System.out.println(nome + " vendendo...");
    }
    
    public double getSalarioTotal() {
        return getSalario() + comissao;
    }
}
```

### 📁 Exercício 3: Polimorfismo com Herança

```java
// Lista de funcionários (inclui gerente e vendedor)
List<Funcionario> equipe = new ArrayList<>();
equipe.add(new Gerente("Maria", 10000, "TI"));
equipe.add(new Vendedor("João", 5000, 2000));
equipe.add(new Funcionario("Pedro", 3000));

// Polimorfismo: cada um trabalha diferente
for (Funcionario f : equipe) {
    f.trabalhar(); // Chama método específico de cada um
}
```
