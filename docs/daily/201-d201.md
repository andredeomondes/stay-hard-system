# Dia 201 — Carreira: Roadmap Pessoal

## Conceito do Dia

Defina seu roadmap de carreira de 1, 3 e 5 anos.

---

## Template de Roadmap

```markdown
# Meu Roadmap de Carreira

## Agora (0-1 ano)
**Posição:** Desenvolvedor Jr/Pleno

**Foco:**
- [ ] Dominar fundamentos (Java, Spring, React)
- [ ] Entregar projetos de valor para empresa
- [ ] Aprender sobre infraestrutura e DevOps
- [ ] Contribuir para open source

**Métricas:**
- Código: >80% coverage
- PRs: 2+ por semana
- Features: 4+ por sprint

## Curto Prazo (1-3 anos)
**Posição:** Desenvolvedor Pleno/Sênior

**Foco:**
- [ ] Arquitetura de sistemas
- [ ] Liderança técnica de projetos
- [ ] Mentorar desenvolvedores Jr
- [ ] Conhecimento de múltiplas tecnologias
- [ ] English fluency (TOEFL/IELTS)

**Métricas:**
- Arquitetou 3+ sistemas
- Mentorou 2+ devs
- Tech talks: 4+/ano
- Article/posts: 12+/ano

## Médio Prazo (3-5 anos)
**Posição:** Tech Lead / Staff Engineer

**Foco:**
- [ ] Estratégia técnica
- [ ] Influenciar roadmap de produto
- [ ] Carreira de IC (Individual Contributor) ou Manager
- [ ] Networking na comunidade
- [ ] Public speaking

**Métricas:**
- Dirigiu 5+ engenheiros
- Decisões arquiteturais implementadas
- Palestras em conferências
- 1000+ seguidores profissionais
```

## Plano de 90 dias

```markdown
# Plano de 90 dias

## Mês 1: Fundamentos
- [ ] Configurar ambiente de desenvolvimento
- [ ] Ler 2 livros técnicos
- [ ] Completar 50 problemas em LeetCode

## Mês 2: Expansão
- [ ] Contribuir para 1 projeto open source
- [ ] Deployar primeira feature em produção
- [ ] Mentorar 1 desenvolvedor Jr

## Mês 3: Visibilidade
- [ ] Escrever 2 artigos técnicos
- [ ] Dar 1 talk interno
- [ ] Revisar e atualizar plano para próximo quarter
```

---

## Checkpoints Trimestrais

```
Q1: Revisar progresso, ajustar metas
Q2: Avaliação de performance
Q3: Planejamento de promoções
Q4: Retroalimentação anual
```
