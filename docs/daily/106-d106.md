## Flexbox Layout

```css
/* Layout principal */
.container {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

main {
    flex: 1;
    padding: 2rem 0;
}

/* Cards em grid */
.cards-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}

.card {
    flex: 1 1 300px;
}

/* Formulário em linha */
.form-row {
    display: flex;
    gap: 0.5rem;
    flex-wrap: wrap;
}

.form-row input,
.form-row select {
    flex: 1;
    min-width: 120px;
}
```

## Responsivo com Media Queries

```css
/* Mobile first (default) */
/* Desktop (> 768px) */
@media (min-width: 768px) {
    .container {
        max-width: 800px;
        margin: 0 auto;
    }
    
    .cards-grid {
        display: grid;
        grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
    }
}

/* Tablet */
@media (min-width: 1024px) {
    .container {
        max-width: 1000px;
    }
    
    .sidebar {
        width: 250px;
        flex-shrink: 0;
    }
    
    .layout {
        display: flex;
    }
}

/* Mobile (< 768px) */
@media (max-width: 767px) {
    .form-row {
        flex-direction: column;
    }
    
    .btn {
        width: 100%;
        justify-content: center;
    }
}
```
