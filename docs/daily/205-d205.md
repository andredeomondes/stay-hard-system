# Dia 205 — Saude: Equilíbrio Vida-Trabalho

## Conceito do Dia

Sustentabilidade é crucial. Cuidar da saúde previne burnout.

---

## Checklist de Bem-estar

```markdown
## Físico
- [ ] 7-8 horas de sono
- [ ] Exercício 3-5x/semana
- [ ] Alimentação balanceada
- [ ] Hidratação adequada
- [ ] Pausas regulares (Pomodoro)

## Mental
- [ ] Meditação/mindfulness
- [ ] Ler livros não-técnicos
- [ ] Hobby fora de tecnologia
- [ ] Férias reais (sem email)

## Social
- [ ] Tempo com família/amigos
- [ ] Networking regular
- [ ] Mentoria (dar e receber)

## Financeiro
- [ ] Fundo de emergência (6 meses)
- [ ] Investimentos (após emergency fund)
- [ ] Orçamentação
```

## Sinais de Burnout

```
⚠️ CRÍTICO ⚠️
- Exaustão constante
- Cinismo em relação ao trabalho
- Dificuldade de concentração
- Insônia
- Irritabilidade frequente
- Tarefas levam mais tempo
- Erros aumentaram
```

## Prevenção

```markdown
## Limites Saudáveis

1. **Horário de trabalho definido**
   - Não responder emails após 19h
   - Fins de semana sem trabalho

2. **Boundaries claros**
   - "Preciso de foco das 8-10h"
   - Reuniões em batch (tarde)

3. **Micro-hábitos**
   - Pausa de 5 min a cada 50 min
   - Alongamento a cada 2h
   - Caminhada no almoço

4. **Desconexão intencional**
   - Férias com no-phone policy
   - Digital detox weekly
```

## Recuperação de Burnout

```markdown
## Se você está em burnout

1. **Reconhecer** - Não ignore os sinais
2. **Comunicar** - Falar com manager/HR
3. **Reducao** - Menos horas ou tarefas
4. **Suporte** - Therapy/coaching
5. **Avaliar** - Este trabalho vale a pena?

Lembre-se: Você é mais valioso saudável.
```
