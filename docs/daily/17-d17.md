### 📁 Exercício 1: HashSet Básico

```java
package com.estudo;

import java.util.*;

public class SetDemo {
    public static void main(String[] args) {
        // Criar set
        Set<String> linguagens = new HashSet<>();
        
        // Adicionar
        linguagens.add("Java");
        linguagens.add("Python");
        linguagens.add("Java"); // Ignorado! Duplicado.
        
        System.out.println(linguagens); // [Java, Python]
        System.out.println("Tamanho: " + linguagens.size()); // 2
        
        // contains
        boolean temJava = linguagens.contains("Java");
        boolean temRuby = linguagens.contains("Ruby");
        
        // Remover
        linguagens.remove("Python");
        
        // Iterar
        for (String lang : linguagens) {
            System.out.println(lang);
        }
    }
}
```

**Checklist:**
- [ ] Criar HashSet
- [ ] Adicionar duplicatas (verificar que ignora)
- [ ] contains()
- [ ] remove()
- [ ] Iterar

### 📁 Exercício 2: Set vs List

```java
// LIST: Permite duplicatas, mantém ordem
List<String> lista = new ArrayList<>();
lista.add("A");
lista.add("A");
lista.add("B");
System.out.println(lista); // [A, A, B] - size = 3

// SET: Não permite duplicatas, não mantém ordem
Set<String> set = new HashSet<>();
set.add("A");
set.add("A");
set.add("B");
System.out.println(set); // [A, B] - size = 2
```

**Checklist:**
- [ ] Comparar List vs Set
- [ ] Verificar que Set ignora duplicatas
- [ ] Entender que Set não mantém ordem

### 📁 Exercício 3: LinkedHashSet (Mantém ordem)

```java
// HashSet: Não mantém ordem
Set<String> hashSet = new HashSet<>();
hashSet.add("Terceiro");
hashSet.add("Primeiro");
hashSet.add("Segundo");
System.out.println(hashSet); // Ordem aleatória

// LinkedHashSet: Mantém ordem de inserção
Set<String> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add("Terceiro");
linkedHashSet.add("Primeiro");
linkedHashSet.add("Segundo");
System.out.println(linkedHashSet); // [Terceiro, Primeiro, Segundo]

// TreeSet: Mantém ordem natural (alfabética/numérica)
Set<String> treeSet = new TreeSet<>();
treeSet.add("Terceiro");
treeSet.add("Primeiro");
treeSet.add("Segundo");
System.out.println(treeSet); // [Primeiro, Segundo, Terceiro]
```

**Checklist:**
- [ ] HashSet (sem ordem)
- [ ] LinkedHashSet (ordem de inserção)
- [ ] TreeSet (ordem natural)

### 📁 Exercício 4: Remover Duplicatas de List

```java
// Problema: List com duplicatas
List<Integer> comDup = List.of(1, 2, 2, 3, 1, 4, 3, 5);

// Solução: Usar Set para remover duplicatas
List<Integer> semDup = new ArrayList<>(new LinkedHashSet<>(comDup));
System.out.println(semDup); // [1, 2, 3, 4, 5]

// Contar palavras únicas
String texto = "java java python ruby java";
String[] palavras = texto.split(" ");
Set<String> unicas = new HashSet<>(Arrays.asList(palavras));
System.out.println("Palavras únicas: " + unicas.size()); // 3
```

**Checklist:**
- [ ] Remover duplicatas
- [ ] Contar únicos
