# Dia 191 — NoSQL: MongoDB

## Conceito do Dia

**MongoDB** é banco de dados de documentos. Schema flexível, ideal para prototipagem.

```yaml
  image: mongo:7

    - "27017:27017"
  environment:
    - MONGO_INITDB_ROOT_USERNAME=admin

```

## Spring Data MongoDB

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-mongodb</artifactId>
</dependency>
```

## Document

```java
@Document(collection = "tarefas")
public class TarefaDocument {
    
    @Id
    private String id;
    
    private String nome;
    private String descricao;
    
    @Field("cat")
    private Categoria categoria;
    
    private Prioridade prioridade;
    
    @Indexed
    private boolean concluida;
    
    private List<String> tags;
    
    private Map<String, Object> metadados;
    
    @CreatedDate
    private LocalDateTime criadoEm;
    
    @LastModifiedDate
    private LocalDateTime atualizadoEm;
}
```

## Repository

```java
@Repository
public interface TarefaMongoRepository extends MongoRepository<TarefaDocument, String> {
    
    List<TarefaDocument> findByCategoria(Categoria categoria);
    
    List<TarefaDocument> findByConcluidaFalse();
    
    List<TarefaDocument> findByTagsContaining(String tag);
    
    @Query("{'categoria': ?0, 'concluida': false}")
    List<TarefaDocument> findPendentesPorCategoria(Categoria cat);
}
```

## Embedded Documents

```java
@Document
public class ProjetoDocument {
    
    @Id
    private String id;
    private String nome;
    
    @Field("tarefas")
    private List<TarefaDocument> tarefas; // Embedded!
    
    private int getTotalTarefas() {
        return tarefas.size();
    }
}
```
