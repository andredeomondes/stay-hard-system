## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**SRP — Single Responsibility Principle**

### Definição em 1 frase:
Uma classe deve ter apenas uma razão para mudar.

### Exemplo:
```
❌ Classe Deus (muda por vários motivos)
public class Usuario {
    void salvar();        // Persistência
    void enviarEmail();   // Email
    void gerarRelatorio(); // Relatório
    void validar();      // Validação
}

✅ Responsabilidades separadas
public class Usuario {}
public class UsuarioRepository {}
public class EmailService {}
public class RelatorioService {}
public class UsuarioValidator {}
```

## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: Identificar SRP

```java
// ❌ ANTES: Classe com muitas responsabilidades
public class Pedido {
    public void criarPedido() { }
    public void salvarNoBanco() { }
    public void enviarEmailConfirmacao() { }
    public void gerarPDF() { }
    public void calcularFrete() { }
}

// ✅ DEPOIS: Responsabilidades separadas
public class Pedido {
    private List<ItemPedido> itens;
    private Cliente cliente;
    // Só dados e lógica de negócio
}

public class PedidoRepository {
    public void salvar(Pedido p) { }
    public Pedido buscar(Long id) { }
}

public class EmailService {
    public void enviarConfirmacao(Pedido p) { }
}

public class PDFService {
    public byte[] gerarPedidoPDF(Pedido p) { }
}
```

**Checklist:**
- [ ] Identificar responsabilidades
- [ ] Separar classes
- [ ] Cada classe com um propósito

### 📁 Exercício 2: Refatorar

```java
// ❌ ANTES
public class UsuarioService {
    public void criar(Usuario u) {
        // valida
        // salva
        // loga
        // envia email
    }
}

// ✅ DEPOIS
public class UsuarioService {
    private UsuarioValidator validator;
    private UsuarioRepository repository;
    private LoggerService logger;
    private EmailService emailService;
    
    public void criar(Usuario u) {
        validator.validar(u);
        repository.salvar(u);
        logger.info("Usuario criado: " + u.getId());
        emailService.enviarBoasVindas(u);
    }
}
```
