### 📁 Exercício 1: Multi-Client Server

**Objetivo:** Servidor que aceita múltiplos clientes

**Arquivos:**
- [ ] `study-projects/fase0-java/tcp-multi-server/src/main/java/com/estudo/Server.java`

**Código:**
```java
package com.estudo;

import java.io.*;
import java.net.*;

public class Server {
    public static void main(String[] args) throws IOException {
        ServerSocket serverSocket = new ServerSocket(8080);
        System.out.println("Servidor ouvindo na porta 8080...");
        
        while (true) {
            // TODO 1: Aceitar novo cliente
            Socket clientSocket = serverSocket.accept();
            System.out.println("Novo cliente: " + clientSocket.getInetAddress());
            
            // TODO 2: Criar thread para tratar cliente
            new Thread(() -> {
                try {
                    tratarCliente(clientSocket);
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }).start();
        }
    }
    
    private static void tratarCliente(Socket socket) {
        try {
            BufferedReader in = new BufferedReader(
                new InputStreamReader(socket.getInputStream())
            );
            PrintWriter out = new PrintWriter(
                socket.getOutputStream(), true
            );
            
            String linha;
            while ((linha = in.readLine()) != null) {
                System.out.println("Cliente disse: " + linha);
                out.println("Eco: " + linha);
            }
            
            socket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```

### 📁 Exercício 2: Port Scanner Simples

**Objetivo:** Verificar quais portas estão abertas

**Arquivos:**
- [ ] `study-projects/fase0-java/port-scanner/src/main/java/com/estudo/PortScanner.java`

**Código:**
```java
package com.estudo;

import java.net.*;
import java.io.*;

public class PortScanner {
    public static void main(String[] args) {
        String host = "localhost";
        
        System.out.println("Escaneando portas em " + host + "...");
        
        // TODO 1: Testar portas comuns
        int[] portas = {80, 443, 8080, 3306, 5432, 27017, 6379};
        
        for (int porta : portas) {
            // TODO 2: Tentar conectar
            try (Socket socket = new Socket()) {
                socket.connect(new InetSocketAddress(host, porta), 1000);
                System.out.println("Porta " + porta + ": ✅ ABERTA");
            } catch (IOException e) {
                System.out.println("Porta " + porta + ": ❌ FECHADA");
            }
        }
        
        System.out.println("Escaneamento completo!");
    }
}
```

## 📝 RESUMO TEÓRICO

### TCP vs UDP:

```
TCP:
✅ Confiável (garante entrega)
✅ Ordenado (na ordem correta)
✅ Orientado a conexão (handshake)
⚠️ Mais lento

UDP:
❌ Não confiável (pode perder)
❌ Não ordenado
❌ Sem conexão
⚡ Mais rápido
```

### Portas comuns:

| Porta | Serviço |
|-------|---------|
| 80 | HTTP |
| 443 | HTTPS |
| 22 | SSH |
| 21 | FTP |
| 25 | SMTP (email) |
| 3306 | MySQL |
| 5432 | PostgreSQL |
| 27017 | MongoDB |
| 6379 | Redis |
| 8080 | Tomcat/HTTP alternativo |
