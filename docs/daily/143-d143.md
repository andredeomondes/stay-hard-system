# Dia 143 — IA: RAG - Retrieval Augmented Generation

## Conceito do Dia

**RAG** combina busca em documentos com LLMs. O modelo responde baseado em conteúdo específico, não apenas conhecimento geral.

## Arquitetura RAG

```
1. INDEXING (offline)
   Documentos → Chunking → Embeddings → Vector DB
   
2. QUERYING (online)
   Query → Embedding → Vector Search → Context → LLM → Response
```

## Indexing

```java
@Service
@RequiredArgsConstructor
public class DocumentIndexingService {
    
    private final VectorStore vectorStore;
    private final DocumentReader documentReader;
    
    public void indexarDocumento(String conteudo, String metadados) {
        // Chunking
        List<String> chunks = chunkText(conteudo, 500);
        
        List<Document> documents = chunks.stream()
            .map(chunk -> Document.builder()
                .text(chunk)
                .metadata(Map.of("source", metadados))
                .build())
            .toList();
        
        // Salvar no Vector Store
        vectorStore.write(documents);
    }
    
    private List<String> chunkText(String text, int maxTokens) {
        // Simples: separar por parágrafos
        return Arrays.stream(text.split("\n\n"))
            .filter(p -> p.length() > 50)
            .toList();
    }
}
```

## Querying

```java
@Service
@RequiredArgsConstructor
public class RAGService {
    
    private final ChatClient chatClient;
    private final VectorStore vectorStore;
    
    public String perguntar(String pergunta) {
        // 1. Buscar contexto relevante
        List<Document> contexto = vectorStore.similaritySearch(
            SearchRequest.builder()
                .query(pergunta)
                .topK(3)
                .build()
        );
        
        // 2. Construir prompt com contexto
        String contextoTexto = contexto.stream()
            .map(Document::getText)
            .collect(Collectors.joining("\n\n"));
        
        String prompt = """
            Baseie sua resposta APENAS no seguinte contexto:
            
            Contexto:
            %s
            
            Pergunta: %s
            
            Se o contexto não tiver informação suficiente, diga que não sei.
            """.formatted(contextoTexto, pergunta);
        
        // 3. Chamar LLM
        return chatClient.call(prompt);
    }
}
```
