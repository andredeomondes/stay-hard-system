## DALL-E 3

```java
@Service
@RequiredArgsConstructor
public class ImageService {
    
    private final ImageModel imageModel;
    
    public byte[] gerarThumbnail(String tarefa) {
        String prompt = """
            Minimalist productivity app thumbnail for a task.
            Clean flat design, soft shadows, modern style.
            Task theme: %s
            
            Style: Clean, professional, modern flat design
            Colors: Blue and white theme
            Size: Square format, 1024x1024
            """.formatted(tarefa);
        
        ImageResponse response = imageModel.call(
            new ImagePrompt(prompt)
                .withModel(ImageModel.DALLE_3)
                .withResolution(new Resolution(1024, 1024))
                .withStyle(ImageOptions.STYLE_VIVID)
        );
        
        return response.getResult().getImage();
    }
}
```

## Uso no Stay Hard

```java
@Service
@RequiredArgsConstructor
public class GamificacaoService {
    
    private final ImageService imageService;
    
    public String gerarBadge(Long tarefaId) {
        Tarefa tarefa = repository.findById(tarefaId).orElseThrow();
        
        String descricao = switch (tarefa.getPrioridade()) {
            case ALTA -> "Epic quest marker, golden border";
            case MEDIA -> "Quest marker, silver border";
            case BAIXA -> "Simple task marker";
        };
        
        String prompt = """
            Achievement badge icon for task completion.
            Theme: %s
            %s
            
            Style: Pixel art, retro game aesthetic
            Size: 256x256
            """.formatted(tarefa.getNome(), descricao);
        
        return imageService.gerarBadge(prompt);
    }
}
```

## Front-end

```jsx
// Componente de badge
function Badge({ tarefa }) {
    const [imageUrl, setImageUrl] = useState(null);
    
    useEffect(() => {
        api.getBadge(tarefa.id)
            .then(url => setImageUrl(url));
    }, [tarefa.id]);
    
    return imageUrl ? (
        <img src={imageUrl} alt="Badge" className="badge" />
    ) : (
        <div className="badge-placeholder">🏆</div>
    );
}
```
