## TarefaDTO com HATEOAS

```java
@EntityModels
public class TarefaDTO extends RepresentationModel<TarefaDTO> {
    
    private Long id;
    private String nome;
    private Categoria categoria;
    private boolean concluida;
    
    public static TarefaDTO of(Tarefa tarefa) {
        return new TarefaDTO(tarefa);
    }
}
```

## Controller com Links

```java
@RestController
@RequestMapping("/api/tarefas")
@RequiredArgsConstructor
public class TarefaController {
    
    private final TarefaService service;
    
    @GetMapping("/{id}")
    public ResponseEntity<TarefaDTO> buscarPorId(@PathVariable Long id) {
        return service.buscarPorId(id)
            .map(tarefa -> {
                TarefaDTO dto = TarefaDTO.of(tarefa);
                
                dto.add(linkTo(methodOn(TarefaController.class).buscarPorId(id)).withSelfRel());
                dto.add(linkTo(methodOn(TarefaController.class).listar()).withRel("tarefas"));
                dto.add(linkTo(methodOn(TarefaController.class).deletar(id))
                    .withRel("deletar").withType("DELETE"));
                
                return dto;
            })
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }
}
```
