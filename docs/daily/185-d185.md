# Dia 185 — GraphQL: Introdução

## Conceito do Dia

**GraphQL** é alternativa a REST. Cliente pede exatamente os dados que precisa.

---

## Por que GraphQL?

```
REST: GET /api/tarefas/1 → retorna tudo
GraphQL: pede só nome e categoria
```

## Dependência

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-graphql</artifactId>
</dependency>
```

## Schema

```graphql
type Query {
    tarefas: [Tarefa!]!
    tarefa(id: ID!): Tarefa
    tarefasPorCategoria(categoria: Categoria!): [Tarefa!]!
}

type Mutation {
    criarTarefa(input: CriarTarefaInput!): Tarefa!
    atualizarTarefa(id: ID!, input: AtualizarTarefaInput!): Tarefa!
    deletarTarefa(id: ID!): Boolean!
}

type Tarefa {
    id: ID!
    nome: String!
    categoria: Categoria!
    prioridade: Prioridade!
    concluida: Boolean!
    criadoEm: DateTime!
    projeto: Projeto
}

enum Categoria {
    ESTUDO
    TRABALHO
    SAUDE
    LAZER
}

enum Prioridade {
    BAIXA
    MEDIA
    ALTA
}

input CriarTarefaInput {
    nome: String!
    categoria: Categoria!
    prioridade: Prioridade!
}
```

## Controller

```java
@Controller
public class TarefaGraphQLController {
    
    @QueryMapping
    public List<Tarefa> tarefas() {
        return service.listarTodas();
    }
    
    @QueryMapping
    public Tarefa tarefa(@Argument Long id) {
        return service.buscarPorId(id).orElse(null);
    }
    
    @MutationMapping
    public Tarefa criarTarefa(@Argument CriarTarefaInput input) {
        return service.criar(input);
    }
}
```

## Cliente

```javascript
// React Query com GraphQL
const GET_TAREFAS = gql`
  query GetTarefas($categoria: Categoria) {
    tarefas(categoria: $categoria) {
      id
      nome
      concluida
    }
  }
`;

function Tarefas() {
  const { data } = useQuery(GET_TAREFAS, {
    variables: { categoria: 'ESTUDO' }
  });
  
  return data?.tarefas.map(t => <TarefaItem key={t.id} {...t} />);
}
```
