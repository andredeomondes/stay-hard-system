# Dia 57 — Filters e listeners

## Conceito do Dia

**Filter**: intercepta requisições antes de chegar ao Servlet (类似中间件/middleware).
**Listener**: responde a eventos do ciclo de vida da aplicação.

## Filter de Logging

```java
@WebFilter("/*")
public class LoggingFilter implements Filter {
    
    @Override
    public void doFilter(ServletRequest req, ServletResponse resp, FilterChain chain) 
            throws IOException, ServletException {
        
        HttpServletRequest httpReq = (HttpServletRequest) req;
        long antes = System.currentTimeMillis();
        
        System.out.println(">>> " + httpReq.getMethod() + " " + httpReq.getRequestURI());
        
        chain.doFilter(req, resp); // continua para o próximo filter/servlet
        
        long depois = System.currentTimeMillis();
        System.out.println("<<< " + (depois - antes) + "ms");
    }
    
    @Override
    public void init(FilterConfig filterConfig) {}
    
    @Override
    public void destroy() {}
}
```

## Filter de Autenticação

```java
@WebFilter("/admin/*")
public class AutenticacaoFilter implements Filter {
    
    @Override
    public void doFilter(ServletRequest req, ServletResponse resp, FilterChain chain) 
            throws IOException, ServletException {
        
        HttpServletRequest httpReq = (HttpServletRequest) req;
        HttpServletResponse httpResp = (HttpServletResponse) resp;
        
        String token = httpReq.getHeader("Authorization");
        
        if (token == null || !token.startsWith("Bearer ")) {
            httpResp.setStatus(401);
            httpResp.getWriter().write("{\"erro\": \"Não autorizado\"}");
            return;
        }
        
        // Validar token...
        
        chain.doFilter(req, resp);
    }
}
```

## Listener de Inicialização

```java
@WebListener
public class AppListener implements ServletContextListener {
    
    @Override
    public void contextInitialized(ServletContextEvent sce) {
        System.out.println(">>> Aplicação iniciada!");
        
        // Inicializar recursos (conexão DB, etc)
    }
    
    @Override
    public void contextDestroyed(ServletContextEvent sce) {
        System.out.println(">>> Aplicação finalizada!");
        
        // Liberar recursos
    }
}
```
