### рҹ“Ғ ExercГӯcio 5: MГәltiplos JOINs

```sql
-- Tabelas:
-- usuarios (id, nome)
-- pedidos (id, usuario_id, produto_id)
-- produtos (id, nome, preco)

-- Juntar 3 tabelas
SELECT 
    u.nome,
    p.nome AS produto,
    pr.preco
FROM usuarios u
INNER JOIN pedidos p ON u.id = p.usuario_id
INNER JOIN produtos pr ON p.produto_id = pr.id
WHERE u.ativo = TRUE
ORDER BY pr.preco DESC;
```

### рҹ“Ғ ExercГӯcio 6: JOIN com AgregaГ§ГЈo

```sql
-- Contar pedidos por usuГЎrio
SELECT 
    u.nome,
    COUNT(p.id) AS total_pedidos,
    SUM(pr.preco) AS valor_total
FROM usuarios u
LEFT JOIN pedidos p ON u.id = p.usuario_id
LEFT JOIN produtos pr ON p.produto_id = pr.id
GROUP BY u.id, u.nome
ORDER BY valor_total DESC;
```
