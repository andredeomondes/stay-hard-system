## Estrutura do Tomcat

```
tomcat/
├── bin/           # Scripts (startup.bat, shutdown.bat)
├── conf/          # Configurações (server.xml, web.xml)
├── lib/           # JARs do Tomcat
├── logs/          # Logs da aplicação
├── webapps/       # Onde ficam suas aplicações
│   ├── minha-app/
│   └── minha-app.war
└── temp/          # Arquivos temporários
```

---

## Deploy de uma aplicação

1. **Criar projeto como WAR:**
```xml
<packaging>war</packaging>
```

2. **Criar classe que estende `HttpServlet`:**
```java
public class TarefaServlet extends HttpServlet { }
```

3. **Compilar e copiar para webapps:**
```bash
mvn clean package
cp target/minha-app.war $CATALINA_HOME/webapps/
```

4. **Acessar:** `http://localhost:8080/minha-app/api/tarefas`

---

## Tomcat com Maven Plugin

```xml
<plugin>
    <groupId>org.apache.tomcat.maven</groupId>
    <artifactId>tomcat7-maven-plugin</artifactId>
    <version>2.2</version>
    <configuration>
        <port>8080</port>
        <path>/stay-hard</path>
    </configuration>
</plugin>
```

```bash
mvn tomcat7:run
```
