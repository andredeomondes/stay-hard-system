# Dia 61 — Spring Boot: Introdução e Setup

## Conceito do Dia

**Spring Boot** simplifica o desenvolvimento Java EE. Ele configura automaticamente o Tomcat, adiciona convenções, e permite rodar uma aplicação com UM arquivo JAR.

"Write less, do more."

## HelloController

```java
@RestController
@RequestMapping("/api")
public class HelloController {
    
    @GetMapping("/ola")
    public String ola() {
        return "Olá! Spring Boot está funcionando!";
    }
    
    @GetMapping("/hora")
    public Map<String, String> hora() {
        return Map.of(
            "hora", LocalTime.now().toString(),
            "mensagem", "Spring Boot é show!"
        );
    }
}
```

---

## Rodar

```bash
mvn spring-boot:run
# ou
./mvnw spring-boot:run
```

Acessar: `http://localhost:8080/api/ola`
