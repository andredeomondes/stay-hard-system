## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**DRY — Don't Repeat Yourself**

### Definição em 1 frase:
Não repita código. Se o mesmo código aparece em mais de um lugar, extraia para um único local.

### 📁 Exercício 1: Identificar duplicação

```java
// ❌ ANTES: Código duplicado
public class UsuarioService {
    public void criar(Usuario u) {
        if (u.getNome() == null || u.getNome().isBlank()) {
            throw new IllegalArgumentException("Nome inválido");
        }
        // criar...
    }
    
    public void atualizar(Usuario u) {
        if (u.getNome() == null || u.getNome().isBlank()) {
            throw new IllegalArgumentException("Nome inválido"); // DUPLICADO!
        }
        // atualizar...
    }
}

// ✅ DEPOIS: Extrair para método
public class UsuarioValidator {
    public void validar(Usuario u) {
        if (u.getNome() == null || u.getNome().isBlank()) {
            throw new IllegalArgumentException("Nome inválido");
        }
    }
}

public class UsuarioService {
    private UsuarioValidator validator;
    
    public void criar(Usuario u) {
        validator.validar(u);
        // criar...
    }
    
    public void atualizar(Usuario u) {
        validator.validar(u);
        // atualizar...
    }
}
```

**Checklist:**
