# Dia 65 — Spring Boot: JSON com Jackson

## Conceito do Dia

O Spring usa **Jackson** para converter objetos Java ↔ JSON. Anotações do Jackson controlam a serialização.

---

## Onde Estudar

- [Artigo: Jackson Annotations](https://www.baeldung.com/jackson-annotations)
- [Docs: Jackson GitHub](https://github.com/FasterXML/jackson-docs)

---

## Anotações Principais

```java
public class Tarefa {
    
    private Long id;
    
    private String nome;
    
    @JsonIgnore
    private String senha; // não aparece no JSON
    
    @JsonFormat(pattern = "dd/MM/yyyy HH:mm")
    private LocalDateTime criadoEm;
    
    @JsonProperty("ativa")
    private boolean concluida; // renomeia no JSON
}
```

## Resultado

```json
{
  "id": 1,
  "nome": "Estudar Spring",
  "ativa": false,
  "criadoEm": "29/03/2026 14:30"
}
```

## Global Config

```yaml
spring:
  jackson:
    serialization:
      write-dates-as-timestamps: false
    deserialization:
      fail-on-unknown-properties: false
    default-property-inclusion: non_null
```

## ResponseEntity para controle de HTTP

```java
@GetMapping("/{id}")
public ResponseEntity<Tarefa> buscarPorId(@PathVariable Long id) {
    return service.buscarPorId(id)
        .map(ResponseEntity::ok)
        .orElse(ResponseEntity.notFound().build());
}
```
