## Endpoints

```
GET    /api/v1/projetos                    # Listar (paginado)
POST   /api/v1/projetos                    # Criar
GET    /api/v1/projetos/{id}               # Buscar
PUT    /api/v1/projetos/{id}               # Atualizar
DELETE /api/v1/projetos/{id}               # Deletar

GET    /api/v1/tarefas                     # Listar (paginado, filtro)
POST   /api/v1/tarefas                     # Criar
GET    /api/v1/tarefas/{id}                # Buscar
PUT    /api/v1/tarefas/{id}                # Atualizar
PATCH  /api/v1/tarefas/{id}                # Atualizar parcial
DELETE /api/v1/tarefas/{id}                # Deletar
POST   /api/v1/tarefas/{id}/concluir       # Marcar concluída

GET    /api/v1/tags                        # Listar tags
POST   /api/v1/tags                        # Criar tag
```
