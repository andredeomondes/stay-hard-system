### 📁 Exercício 2: Primeiro Teste

```java
package com.estudo;

import org.junit.jupiter.api.*;

class CalculadoraTest {
    
    @Test
    @DisplayName("Soma de dois números positivos")
    void testSomaDoisNumerosPositivos() {
        // Arrange (dados de entrada)
        int a = 2;
        int b = 3;
        Calculadora calc = new Calculadora();
        
        // Act (ação)
        int resultado = calc.somar(a, b);
        
        // Assert (verificação)
        Assertions.assertEquals(5, resultado);
    }
}
```

**Checklist:**
- [ ] @Test annotation
- [ ] Arrange-Act-Assert
- [ ] assertEquals()
- [ ] `mvn test`

### 📁 Exercício 3: @BeforeEach e @AfterEach

```java
class CalculadoraTest {
    
    private Calculadora calc;
    
    @BeforeEach
    void setup() {
        // Roda ANTES de cada teste
        calc = new Calculadora();
        System.out.println("Setup executado");
    }
    
    @AfterEach
    void teardown() {
        // Roda DEPOIS de cada teste
        System.out.println("Teardown executado");
    }
    
    @Test
    void testSoma() {
        assertEquals(5, calc.somar(2, 3));
    }
    
    @Test
    void testSubtracao() {
        assertEquals(1, calc.subtrair(3, 2));
    }
}
```

**Checklist:**
- [ ] @BeforeEach
- [ ] @AfterEach
- [ ] Entender ciclo de vida

### 📁 Exercício 4: Assertions Comuns

```java
import static org.junit.jupiter.api.Assertions.*;

// assertEquals - Verifica igualdade
assertEquals(5, calc.somar(2, 3));

// assertTrue / assertFalse
assertTrue(calc.ehPar(4));
assertFalse(calc.ehPar(3));

// assertNull / assertNotNull
assertNull(objetoNulo);
assertNotNull(objetoValido);

// assertSame / assertNotSame (verifica referência)
assertSame(obj1, obj2); // Mesma instância

// assertArrayEquals
assertArrayEquals(new int[]{1,2,3}, array);

// assertThrows - Exceção
assertThrows(ArithmeticException.class, 
    () -> calc.dividir(1, 0)
);

// assertDoesNotThrow - Não lança exceção
assertDoesNotThrow(() -> calc.somar(1, 2));
```

**Checklist:**
- [ ] assertEquals
- [ ] assertTrue/assertFalse
- [ ] assertNull/assertNotNull
- [ ] assertThrows
- [ ] assertDoesNotThrow
