# Dia 84 — REST API: PUT vs PATCH

## Conceito do Dia

**PUT** substitui o recurso inteiro. **PATCH** atualiza apenas campos enviados (parcial).

---

## PUT: Atualização Completa

```java
@PutMapping("/{id}")
public ResponseEntity<Tarefa> atualizar(
        @PathVariable Long id,
        @Valid @RequestBody AtualizarTarefaRequest request) {
    
    return service.atualizar(id, request)
        .map(ResponseEntity::ok)
        .orElse(ResponseEntity.notFound().build());
}

// AtualizarTarefaRequest (todos os campos)
public record AtualizarTarefaRequest(
    String nome,
    Categoria categoria,
    Prioridade prioridade,
    boolean concluida
) {}

// Se enviar {"nome": "Novo nome"} → campos não enviados viram null!
```

## PATCH: Atualização Parcial

```java
@PatchMapping("/{id}")
public ResponseEntity<Tarefa> atualizarParcial(
        @PathVariable Long id,
        @RequestBody Map<String, Object> updates) {
    
    return service.atualizarParcial(id, updates)
        .map(ResponseEntity::ok)
        .orElse(ResponseEntity.notFound().build());
}
```

```java
public Optional<Tarefa> atualizarParcial(Long id, Map<String, Object> updates) {
    return repository.findById(id)
        .map(tarefa -> {
            updates.forEach((campo, valor) -> {
                switch (campo) {
                    case "nome" -> tarefa.setNome((String) valor);
                    case "prioridade" -> tarefa.setPrioridade(Prioridade.valueOf((String) valor));
                    case "concluida" -> tarefa.setConcluida((Boolean) valor);
                    case "categoria" -> tarefa.setCategoria(Categoria.valueOf((String) valor));
                }
            });
            return repository.save(tarefa);
        });
}
```

## Exemplo

```bash
# PUT - substitui tudo
curl -X PUT /api/tarefas/1 \
  -H "Content-Type: application/json" \
  -d '{"nome":"Novo","categoria":"LAZER","prioridade":"BAIXA","concluida":false}'
# Resultado: Todos os campos atualizados

# PATCH - atualiza só o que enviou
curl -X PATCH /api/tarefas/1 \
  -H "Content-Type: application/json" \
  -d '{"concluida": true}'
# Resultado: só concluida muda
```
