## Naming Conventions

```java
public interface TarefaRepository extends JpaRepository<Tarefa, Long> {
    
    // findBy + nome do campo
    List<Tarefa> findByNome(String nome);
    
    // Contém (LIKE %termo%)
    List<Tarefa> findByNomeContaining(String termo);
    
    // Começa com
    List<Tarefa> findByNomeStartingWith(String prefixo);
    
    // Múltiplos campos (AND)
    List<Tarefa> findByCategoriaAndPrioridade(Categoria cat, Prioridade p);
    
    // OU
    List<Tarefa> findByCategoriaOrPrioridade(Categoria cat, Prioridade p);
    
    // Ordenação
    List<Tarefa> findByConcluidaFalseOrderByCriadoEmDesc();
    
    // Between (para datas)
    List<Tarefa> findByCriadoEmBetween(LocalDateTime ini, LocalDateTime fim);
    
    // Count
    long countByCategoria(Categoria cat);
    
    // Exists
    boolean existsByNome(String nome);
    
    // Delete
    void deleteByCategoria(Categoria cat);
}
```

## Tradução Automática

```
findByNomeContainingIgnoreCase(String termo)
   ↓
SELECT * FROM tarefas WHERE LOWER(nome) LIKE LOWER(CONCAT('%', ?, '%'))
```
