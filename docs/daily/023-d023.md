## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício: TDD com Calculadora

**PASSO 1: Escreva o teste PRIMEIRO (RED)**

```java
class CalculadoraTddTest {
    
    private CalculadoraTDD calc;
    
    @BeforeEach
    void setup() {
        calc = new CalculadoraTDD();
    }
    
    // TESTE 1: Subtração
    @Test
    @DisplayName("Subtração: 5 - 3 = 2")
    void testSubtracao() {
        assertEquals(2, calc.subtrair(5, 3));
    }
    
    // TESTE 2: Multiplicação
    @Test
    @DisplayName("Multiplicação: 4 * 3 = 12")
    void testMultiplicacao() {
        assertEquals(12, calc.multiplicar(4, 3));
    }
    
    // TESTE 3: Divisão
    @Test
    @DisplayName("Divisão: 10 / 2 = 5")
    void testDivisao() {
        assertEquals(5.0, calc.dividir(10, 2));
    }
    
    // TESTE 4: Divisão por zero
    @Test
    @DisplayName("Divisão por zero lança exceção")
    void testDivisaoPorZero() {
        assertThrows(ArithmeticException.class,
            () -> calc.dividir(10, 0)
        );
    }
}
```

**PASSO 2: Implemente o mínimo para passar (GREEN)**

```java
class CalculadoraTDD {
    
    int subtrair(int a, int b) {
        return a - b;
    }
    
    int multiplicar(int a, int b) {
        return a * b;
    }
    
    double dividir(int a, int b) {
        if (b == 0) {
            throw new ArithmeticException("Divisão por zero");
        }
        return (double) a / b;
    }
}
```

**PASSO 3: Refatore (se necessário)**

        }
        return (double) a / b;
    }
}
