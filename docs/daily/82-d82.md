# Dia 82 — REST API: ResponseEntity e HTTP Status

## Conceito do Dia

O Spring usa `ResponseEntity` para controlar status HTTP, headers e body da resposta.

## TarefaController completo

```java
@RestController
@RequestMapping("/api/tarefas")
@RequiredArgsConstructor
public class TarefaController {
    
    private final TarefaService service;
    
    @GetMapping
    public ResponseEntity<List<Tarefa>> listar(
            @RequestParam(required = false) Categoria categoria,
            @RequestParam(defaultValue = "false") boolean pendentes) {
        
        List<Tarefa> tarefas;
        if (pendentes) {
            tarefas = service.buscarPendentes();
        } else if (categoria != null) {
            tarefas = service.buscarPorCategoria(categoria);
        } else {
            tarefas = service.listarTodas();
        }
        
        return ResponseEntity.ok(tarefas);
    }
    
    @GetMapping("/{id}")
    public ResponseEntity<Tarefa> buscarPorId(@PathVariable Long id) {
        return service.buscarPorId(id)
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }
    
    @PostMapping
    public ResponseEntity<Tarefa> criar(@Valid @RequestBody CriarTarefaRequest request) {
        Tarefa tarefa = service.criar(request);
        
        URI location = ServletUriComponentsBuilder
            .fromCurrentRequest()
            .path("/{id}")
            .buildAndExpand(tarefa.getId())
            .toUri();
        
        return ResponseEntity.created(location).body(tarefa);
    }
    
    @PutMapping("/{id}")
    public ResponseEntity<Tarefa> atualizar(
            @PathVariable Long id,
            @Valid @RequestBody AtualizarTarefaRequest request) {
        
        return service.atualizar(id, request)
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }
    
    @DeleteMapping("/{id}")
    public ResponseEntity<Void> deletar(@PathVariable Long id) {
        service.deletar(id);
        return ResponseEntity.noContent().build();
    }
}
```

## HTTP Status Codes comuns

| Código | Uso |
|--------|-----|
| 200 | OK (GET, PUT) |
| 201 | Created (POST) |
| 204 | No Content (DELETE) |
| 400 | Bad Request (validação) |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 409 | Conflict |
| 500 | Internal Server Error |
