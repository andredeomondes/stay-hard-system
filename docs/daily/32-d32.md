### Conceito:
**Encapsulamento — Proteger dados**

### Definição em 1 frase:
Encapsulamento oculta detalhes internos e expõe apenas o necessário.

### Exemplo:
```
Carro: Você aperta acelerador, não precisa saber do motor.
Classe: Você usa getNome(), não precisa saber como é guardado.
```

## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: private + getters/setters

```java
public class Usuario {
    // private = só a própria classe acessa
    private String nome;
    private int idade;
    
    // Construtor
    public Usuario(String nome, int idade) {
        this.nome = nome;
        setIdade(idade); // Usa setter para validar
    }
    
    // Getter (ler)
    public String getNome() {
        return nome;
    }
    
    // Setter (escrever com validação)
    public void setNome(String nome) {
        if (nome == null || nome.isBlank()) {
            throw new IllegalArgumentException("Nome inválido");
        }
        this.nome = nome;
    }
    
    public int getIdade() {
        return idade;
    }
    
    public void setIdade(int idade) {
        if (idade < 0 || idade > 150) {
            throw new IllegalArgumentException("Idade inválida");
        }
        this.idade = idade;
    }
}
```

**Uso:**
```java
Usuario u = new Usuario("André", 25);
u.setNome("Maria"); // Valida
u.setIdade(-5);   // Lança exceção
System.out.println(u.getNome()); // Maria
```

### 📁 Exercício 2: Imutabilidade

```java
public class Endereco {
    private final String rua;
    private final String cidade;
    private final String cep;
    
    public Endereco(String rua, String cidade, String cep) {
        this.rua = rua;
        this.cidade = cidade;
        this.cep = cep;
    }
    
    // Sem setters! final = não muda depois de criado
    
    public String getRua() { return rua; }
    public String getCidade() { return cidade; }
    public String getCep() { return cep; }
}
```
