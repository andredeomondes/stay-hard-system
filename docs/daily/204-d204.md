# Dia 204 — Produtividade: Deep Work

## Conceito do Dia

**Deep Work** é foco intenso sem distrações. Produza mais em menos tempo.

---

## Regras de Cal Newport

```
1. Trabalhe profundamente
2. Embrace boredom
3. Quit social media
4. Drain the shallows
```

## Cronograma

```markdown
## Dia Ideal

06:00 - Wake up, exercise
07:00 - Breakfast, ready
08:00 - Deep Work (2h) ← hardest task
10:00 - Break
10:30 - Deep Work (2h) ← second task
12:30 - Lunch
13:30 - Shallow work (emails, meetings)
15:00 - Deep Work (1.5h)
16:30 - Review, plan next day
17:00 - End of day
```

## Técnicas

### Pomodoro
```
25 min work → 5 min break
4 pomodoros → 15 min break
```

### Time Blocking
```
Calendário:
[08:00-10:00] Deep Work: API Authentication
[10:30-12:30] Deep Work: Database Migration
[14:00-15:30] Meetings
[15:30-17:00] Shallow Work
```


## Estratégias Anti-procrastinação

```markdown
1. **Eat the frog** - Faça a tarefa mais difícil primeiro

2. **2-minute rule** - Se leva < 2 min, faça agora

3. **Implementation intention**
   "Quando X acontecer, vou fazer Y"
   "Quando terminar esta task, vou ao próximo"

4. **Progress over perfection**
   Feito > Perfeito
```
