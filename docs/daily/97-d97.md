# Dia 97 — Spring Security: Refresh Token

## Conceito do Dia

Tokens expiram. **Refresh Token** permite obter um novo access token sem fazer login novamente.

## RefreshToken Entity

```java
@Entity
public class RefreshToken {
    
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    
    @Column(nullable = false, unique = true)
    private String token;
    
    @Column(nullable = false)
    private LocalDateTime expiryDate;
    
    @OneToOne
    @JoinColumn(name = "user_id")
    private User user;
    
    public boolean isExpired() {
        return LocalDateTime.now().isAfter(expiryDate);
    }
}
```

## RefreshTokenService

```java
@Service
@RequiredArgsConstructor
public class RefreshTokenService {
    
    private final RefreshTokenRepository repository;
    
    public String createRefreshToken(User user) {
        RefreshToken token = new RefreshToken();
        token.setUser(user);
        token.setToken(UUID.randomUUID().toString());
        token.setExpiryDate(LocalDateTime.now().plusDays(7));
        
        repository.save(token);
        return token.getToken();
    }
    
    public Optional<RefreshToken> findByToken(String token) {
        return repository.findByToken(token);
    }
    
    public void deleteByUser(User user) {
        repository.deleteByUser(user);
    }
}
```

## AuthController atualizado

```java
public record RefreshRequest(String refreshToken) {}

public record TokenResponse(String accessToken, String refreshToken, long expiresIn) {}

// Refresh endpoint
@PostMapping("/refresh")
public TokenResponse refresh(@RequestBody RefreshRequest request) {
    RefreshToken stored = refreshTokenService.findByToken(request.refreshToken())
        .orElseThrow(() -> new InvalidRefreshTokenException());
    
    if (stored.isExpired()) {
        refreshTokenService.deleteByUser(stored.getUser());
        throw new InvalidRefreshTokenException();
    }
    
    String newAccessToken = jwtService.generateAccessToken(stored.getUser());
    String newRefreshToken = refreshTokenService.createRefreshToken(stored.getUser());
    
    return new TokenResponse(newAccessToken, newRefreshToken, 3600);
}
```

## Fluxo

```
1. Login → access_token + refresh_token
2. access_token expira
3. POST /api/auth/refresh → novo access_token + novo refresh_token
4. Refresh token expira → fazer login novamente
```
