## Dependência

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-openai-spring-boot-starter</artifactId>
</dependency>
```

## Configuração

```yaml
spring:
  application:
    name: stayhard-ai
  
  ai:
    openai:
      api-key: ${OPENAI_API_KEY}
      chat:
        options:
          model: gpt-4o
          temperature: 0.7
```

## Uso

```java
@Service
@RequiredArgsConstructor
public class TarefaIAService {
    
    private final ChatClient chatClient;
    
    public String sugerirMelhorias(String nomeTarefa) {
        String prompt = """
            Analise a seguinte tarefa e sugira melhorias:
            
            Tarefa: %s
            
            Considere:
            1. A tarefa é específica o suficiente?
            2. Pode ser dividida em partes menores?
            3. Tem critérios de sucesso claros?
            
            Responda de forma concisa.
            """.formatted(nomeTarefa);
        
        return chatClient.call(prompt);
    }
    
    public List<String> gerarSubtarefas(String tarefa) {
        String prompt = """
            Divida a seguinte tarefa em 3-5 subtarefas:
            %s
            
            Responda apenas com uma lista numerada.
            """.formatted(tarefa);
        
        return chatClient.call(prompt)
            .lines()
            .toList();
    }
}
```

## Structured Output

```java
@Service
@RequiredArgsConstructor
public class ClassificacaoService {
    
    private final ChatClient chatClient;
    
    public TarefaClassificacao classificar(String descricao) {
        Prompt prompt = new Prompt("Classifique: " + descricao);
        
        return chatClient.call(prompt, TarefaClassificacao.class);
    }
}

public record TarefaClassificacao(
    Categoria categoriaSugerida,
    Prioridade prioridadeSugerida,
    List<String> tagsSugeridas,
    String justificativa
) {}
```
