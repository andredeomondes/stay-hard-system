## Conceitos

| Conceito | Descrição |
|----------|------------|
| Pod | Menor unidade (1+ containers) |
| Deployment | Gerencia Pods |
| Service | Exposição de Pods |
| Ingress | HTTP routing |
| ConfigMap | Configurações |
| Secret | Dados sensíveis |

## Deployment

```yaml
# deployment.yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: tarefa-service
spec:
  replicas: 3
  selector:
    matchLabels:
      app: tarefa-service
  template:
    metadata:
      labels:
        app: tarefa-service
    spec:
      containers:
        - name: tarefa-service
          image: stayhard/tarefa-service:latest
          ports:
            - containerPort: 8080
          env:
            - name: SPRING_PROFILES_ACTIVE
              value: "prod"
            - name: DB_HOST
              valueFrom:
                configMapKeyRef:
                  name: tarefa-config
                  key: db-host
            - name: DB_PASSWORD
              valueFrom:
                secretKeyRef:
                  name: tarefa-secrets
                  key: db-password
          resources:
            requests:
              memory: "256Mi"
              cpu: "250m"
            limits:
              memory: "512Mi"
              cpu: "500m"
          livenessProbe:
            httpGet:
              path: /actuator/health/liveness
              port: 8080
            initialDelaySeconds: 30
          readinessProbe:
            httpGet:
              path: /actuator/health/readiness
              port: 8080
```

## Service

```yaml
# service.yaml
apiVersion: v1
kind: Service
metadata:
  name: tarefa-service
spec:
  selector:
    app: tarefa-service
  ports:
    - port: 80
      targetPort: 8080
  type: ClusterIP
```

## Ingress

```yaml
# ingress.yaml
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: stayhard-ingress
spec:
  rules:
    - host: api.stayhard.com
      http:
        paths:
          - path: /tarefas
            pathType: Prefix
            backend:
              service:
                name: tarefa-service
                port:
                  number: 80
```
