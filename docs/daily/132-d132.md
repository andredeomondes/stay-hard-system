# Dia 132 — API Gateway: Spring Cloud Gateway

## Conceito do Dia

**API Gateway** é ponto único de entrada. Roteia requisições, faz autenticação, rate limiting.

---

## Onde Estudar

- [Docs: Spring Cloud Gateway](https://docs.spring.io/spring-cloud-gateway/docs/current/reference/html/)

---

## Dependência

```xml
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-gateway</artifactId>
</dependency>
```

## application.yml

```yaml
server:
  port: 8080

spring:
  cloud:
    gateway:
      routes:
        - id: auth-service
          uri: http://auth-service:8081
          predicates:
            - Path=/api/auth/**
        
        - id: tarefa-service
          uri: http://tarefa-service:8082
          predicates:
            - Path=/api/tarefas/**
        
        - id: projeto-service
          uri: http://projeto-service:8083
          predicates:
            - Path=/api/projetos/**
      
      globalcors:
        corsConfigurations:
          '[/**]':
            allowedOrigins: "*"
            allowedMethods: "*"
            allowedHeaders: "*"
```

## Filtros Customizados

```java
@Component
public class AuthFilter implements GlobalFilter {
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        String path = exchange.getRequest().getPath().value();
        
        // Pular autenticação para rotas públicas
        if (path.startsWith("/api/auth/")) {
            return chain.filter(exchange);
        }
        
        String authHeader = exchange.getRequest().getHeaders().getFirst("Authorization");
        
        if (authHeader == null || !authHeader.startsWith("Bearer ")) {
            exchange.getResponse().setStatusCode(HttpStatus.UNAUTHORIZED);
            return exchange.getResponse().setComplete();
        }
        
        // Validar token...
        
        return chain.filter(exchange);
    }
}
```
