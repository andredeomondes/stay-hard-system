## Definir Functions

```java
public class TarefaFunctions {
    
    public static final String GET_TAREFA = "getTarefa";
    public static final String LIST_TAREFAS = "listTarefas";
    public static final String CRIAR_TAREFA = "criarTarefa";
    
    @Tool(name = GET_TAREFA, description = "Busca uma tarefa por ID")
    public record GetTarefaTool(
        @ToolParam(description = "ID da tarefa") Long id
    ) {}
    
    @Tool(name = LIST_TAREFAS, description = "Lista tarefas com filtros")
    public record ListTarefasTool(
        @ToolParam(description = "Categoria da tarefa") String categoria,
        @ToolParam(description = "Status de conclusão") Boolean concluida
    ) {}
}
```

## Service com Tool

```java
@Service
@RequiredArgsConstructor
public class TarefaToolService {
    
    private final TarefaRepository repository;
    
    public Tarefa getTarefa(Long id) {
        return repository.findById(id)
            .orElseThrow(() -> new RecursoNaoEncontradoException("tarefa", id));
    }
    
    public List<Tarefa> listTarefas(String categoria, Boolean concluida) {
        if (categoria != null && concluida != null) {
            Categoria cat = Categoria.valueOf(categoria.toUpperCase());
            return repository.findByCategoriaAndConcluida(cat, concluida);
        }
        if (categoria != null) {
            Categoria cat = Categoria.valueOf(categoria.toUpperCase());
            return repository.findByCategoria(cat);
        }
        if (concluida != null) {
            return concluida ? repository.findByConcluidaTrue() 
                            : repository.findByConcluidaFalse();
        }
        return repository.findAll();
    }
}
```

## Controller com Chat

```java
@RestController
@RequestMapping("/api/ai")
@RequiredArgsConstructor
public class AIAssistantController {
    
    private final ChatClient chatClient;
    private final TarefaToolService tarefaTool;
    
    @PostMapping("/chat")
    public String chat(@RequestBody ChatRequest request) {
        String userMessage = request.message();
        
        // Contexto da conversa
        String systemPrompt = """
            Você é um assistente de produtividade. Ajude o usuário a gerenciar tarefas.
            Use as ferramentas disponíveis para buscar informações.
            """;
        
        Prompt prompt = Prompt.builder()
            .system(systemPrompt)
            .user(userMessage)
            .build();
        
        // ChatClient com tools
        return chatClient.prompt(prompt)
            .tools(new Object[] {tarefaTool})
            .call()
            .content();
    }
}
```

## Exemplo de Uso

```
Usuário: Quais são minhas tarefas pendentes de estudo?

LLM: Vou buscar suas tarefas pendentes de estudo.
→ listTarefas(categoria="ESTUDO", concluida=false)
← [Tarefa(id=1, nome="Estudar Spring"), Tarefa(id=3, nome="Revisar Java")]

LLM: Você tem 2 tarefas pendentes de estudo:
1. Estudar Spring
2. Revisar Java
```
