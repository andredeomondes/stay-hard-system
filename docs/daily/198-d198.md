# Dia 198 — Accessibility: A11y

## Conceito do Dia

**Accessibility** garante que todos possam usar sua aplicação, incluindo pessoas com deficiências.

---

## WCAG 2.1 Guidelines

| Princípio | Descrição |
|-----------|-----------|
| Perceivable | Informações são apresentáveis |
| Operable | Interface é operável |
| Understandable | Informação é compreensível |
| Robust | Conteúdo é interpretável |

## Checklist Técnico

```html
<!-- Imagens com alt -->
<img src="grafico.png" alt="Gráfico mostrando aumento de 20% em vendas">

<!-- Form labels -->
<label for="email">Email</label>
<input type="email" id="email" aria-describedby="email-help">
<span id="email-help">Formato: nome@exemplo.com</span>

<!-- Botões com texto -->
<button>
  <span class="visually-hidden">Deletar</span>
  <span aria-hidden="true">🗑️</span>
</button>

<!-- Contraste -->
<!-- Minimum 4.5:1 for text -->
<div style="color: #767676; background: #ffffff;">
  <!-- Violação: contraste insuficiente -->
</div>
```

## ARIA

```html
<!-- Alert para notificações -->
<div role="alert" aria-live="polite">
  Tarefa criada com sucesso!
</div>

<!-- Modal dialog -->
<div role="dialog" aria-modal="true" aria-labelledby="modal-title">
  <h2 id="modal-title">Confirmar exclusão</h2>
  <button>Cancelar</button>
  <button>Confirmar</button>
</div>

<!-- Skip link -->
<a href="#main-content" class="skip-link">
  Pular para conteúdo principal
</a>

<!-- Loading state -->
<div role="status" aria-live="polite">
  Carregando tarefas...
</div>
```

## React A11y

```jsx
import { AriaLabel, VisuallyHidden } from '@radix-ui/react-visually-hidden';

function DeleteButton({ onClick }) {
  return (
    <button 
      onClick={onClick}
      aria-label="Deletar tarefa"
      title="Deletar"
    >
      <VisuallyHidden>Deletar</VisuallyHidden>
      🗑️
    </button>
  );
}

// Form com aria-describedby
<Form>
  <label htmlFor="senha">Senha</label>
  <input 
    id="senha" 
    type="password"
    aria-describedby="senha-hint"
  />
  <span id="senha-hint">
    Mínimo 8 caracteres, incluindo número
  </span>
</Form>
```
