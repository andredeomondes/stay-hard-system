## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: TCP Server

**Objetivo:** Criar um servidor TCP que recebe mensagens

**Arquivos:**
- [ ] `study-projects/fase0-java/tcp-server/src/main/java/com/estudo/Server.java`

**Passo a passo:**

**1.1 — Criar estrutura:**
```
tcp-server/
├── pom.xml
└── src/main/java/com/estudo/
    └── Server.java
```

**1.2 — Criar pom.xml:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<project>
    <modelVersion>4.0.0</modelVersion>
    <groupId>com.estudo</groupId>
    <artifactId>tcp-server</artifactId>
    <version>1.0.0</version>
    
    <properties>
        <maven.compiler.source>21</maven.compiler.source>
        <maven.compiler.target>21</maven.compiler.target>
    </properties>
</project>
```

**1.3 — Implementar Server.java:**
```java
package com.estudo;

import java.io.*;
import java.net.*;

public class Server {
    public static void main(String[] args) throws IOException {
        // TODO 1: Criar ServerSocket na porta 8080
        ServerSocket serverSocket = new ServerSocket(8080);
        System.out.println("Servidor ouvindo na porta 8080...");
        
        // TODO 2: Aguardar conexão (accept)
        Socket clientSocket = serverSocket.accept();
        System.out.println("Cliente conectado!");
        
        // TODO 3: Criar streams para ler e escrever
        BufferedReader in = new BufferedReader(
            new InputStreamReader(clientSocket.getInputStream())
        );
        PrintWriter out = new PrintWriter(
            clientSocket.getOutputStream(), true
        );
        
        // TODO 4: Ler mensagem do cliente
        String mensagem = in.readLine();
        System.out.println("Cliente disse: " + mensagem);
        
        // TODO 5: Responder ao cliente
        out.println("Olá cliente! Recebi sua mensagem.");
        
        // TODO 6: Fechar conexões
        clientSocket.close();
        serverSocket.close();
    }
}
```

**Checklist:**
- [ ] Criar pasta `tcp-server`
- [ ] Criar `pom.xml`
- [ ] Criar estrutura de pacotes
- [ ] Implementar `Server.java`
- [ ] `mvn compile`
- [ ] Testar (manualmente com telnet ou nc)

### 📁 Exercício 2: TCP Client

**Objetivo:** Criar cliente TCP que se conecta ao servidor

**Arquivos:**
- [ ] `study-projects/fase0-java/tcp-client/src/main/java/com/estudo/Client.java`

**Código:**
```java
package com.estudo;

import java.io.*;
import java.net.*;

public class Client {
    public static void main(String[] args) throws IOException {
        // TODO 1: Conectar ao servidor
        Socket socket = new Socket("localhost", 8080);
        System.out.println("Conectado ao servidor!");
        
        // TODO 2: Criar streams
        PrintWriter out = new PrintWriter(
            socket.getOutputStream(), true
        );
        BufferedReader in = new BufferedReader(
            new InputStreamReader(socket.getInputStream())
        );
        
        // TODO 3: Enviar mensagem
        out.println("Olá servidor!");
        System.out.println("Mensagem enviada!");
        
        // TODO 4: Receber resposta
        String resposta = in.readLine();
        System.out.println("Servidor respondeu: " + resposta);
        
        // TODO 5: Fechar conexão
        socket.close();
    }
}
```

**Checklist:**
- [ ] Criar pasta `tcp-client`
- [ ] Criar `pom.xml`
- [ ] Implementar `Client.java`
- [ ] `mvn compile`
