# Dia 150 — Mini-Projeto: IA no Stay Hard

## Conceito do Dia

Integrar IA no Stay Hard: assistente de produtividade, sugestões, e gamificação.

---

## Funcionalidades

### 1. Assistente de Tarefas (Chat)

```
Usuário: "Tenho 5 tarefas urgente. Me ajude a priorizar"
Assistente: "Vamos lá! Com base no impacto e deadline..."

→ usar RAG para contexto de produtividade
→ usar function calling para buscar tarefas
→ sugerir ordem de execução
```

### 2. Divisor de Tarefas

```
Usuário: "Preparar apresentação para cliente"
Assistente: "Vou dividir isso para você!"
→ 1. Pesquisar tema
→ 2. Criar outline
→ 3. Criar slides
→ 4. Revisar conteúdo
→ 5. Preparar backup
```

### 3. Sugestor de Tags

```
Usuário: Cria tarefa "Implementar login com OAuth"
Assistente: "Sugiro adicionar tags: [backend, security, oauth]"
→ usar embeddings para sugerir tags baseadas em histórico
```

### 4. Relatório de Produtividade

```
"Resuma minha semana de produtividade"
→ buscar tarefas concluídas
→ gerar relatório com métricas
→ dar dicas para próxima semana
```

## API Endpoints

```java
@RestController
@RequestMapping("/api/ai")
@RequiredArgsConstructor
public class AIAssistantController {
    
    private final ChatService chatService;
    private final TarefaDivisorService divisorService;
    private final SugestorTagsService sugeridorService;
    
    @PostMapping("/chat")
    public AIResponse chat(@RequestBody ChatRequest request) {
        return chatService.processar(request.message(), request.sessionId());
    }
    
    @PostMapping("/dividir/{tarefaId}")
    public List<Subtarefa> dividirTarefa(@PathVariable Long tarefaId) {
        return divisorService.dividir(tarefaId);
    }
    
    @PostMapping("/sugerir-tags/{tarefaId}")
    public List<String> sugerirTags(@PathVariable Long tarefaId) {
        return sugeridorService.sugerirTags(tarefaId);
    }
}
```

// Chatbot component
function AIAssistant() {
    const [messages, setMessages] = useState([]);
    
    const sendMessage = async (text) => {
        setMessages(prev => [...prev, { role: 'user', content: text }]);
        
        const response = await api.aiChat(text);
        setMessages(prev => [...prev, { role: 'assistant', content: response }]);
    };
    
    return (
        <div className="assistant">
            {messages.map((m, i) => (
                <Message key={i} role={m.role} content={m.content} />
            ))}
            <ChatInput onSend={sendMessage} />
        </div>
    );
}
