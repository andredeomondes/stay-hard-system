## 🎯 O QUE VOCÊ VAI APRENDER HOJE

### Conceito:
**Transactions — Garantir consistência de dados**

### Definição em 1 frase:
Transaction garante que um conjunto de operações seja executado completamente ou não seja executado nada.

### ACID:
```
Atomicidade → Tudo ou nada
Consistência → Estado válido sempre
Isolamento → Operações simultâneas não interferem
Durabilidade → Após commit, fica salvo
```

## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: Transaction Básico

```java
public void transferir(int deId, int paraId, double valor) {
    Connection conn = null;
    try {
        conn = Database.getConnection();
        conn.setAutoCommit(false); // Iniciar transaction
        
        // Debitar
        String debitar = "UPDATE contas SET saldo = saldo - ? WHERE id = ?";
        try (PreparedStatement stmt = conn.prepareStatement(debitar)) {
            stmt.setDouble(1, valor);
            stmt.setInt(2, deId);
            stmt.executeUpdate();
        }
        
        // Creditar
        String creditar = "UPDATE contas SET saldo = saldo + ? WHERE id = ?";
        try (PreparedStatement stmt = conn.prepareStatement(creditar)) {
            stmt.setDouble(1, valor);
            stmt.setInt(2, paraId);
            stmt.executeUpdate();
        }
        
        conn.commit(); // Salvar
        System.out.println("Transferência realizada!");
        
    } catch (SQLException e) {
        if (conn != null) {
            conn.rollback(); // Desfazer tudo
        }
        throw new RuntimeException("Transferência falhou", e);
    } finally {
        if (conn != null) {
            conn.setAutoCommit(true);
            conn.close();
        }
    }
}
```

**Checklist:**
- [ ] setAutoCommit(false)
- [ ] commit()
- [ ] rollback()
- [ ] finally com cleanup

### 📁 Exercício 2: Savepoint

```java
public void operacaoComplexa() {
    try (Connection conn = Database.getConnection()) {
        conn.setAutoCommit(false);
        Savepoint sp1 = conn.setSavepoint("ponto1");
        
        // Operações...
        conn.execute("INSERT...");
        conn.execute("UPDATE...");
        
        Savepoint sp2 = conn.setSavepoint("ponto2");
        conn.execute("DELETE...");
        
        // Rollback para ponto específico
        conn.rollback(sp1); // Desfaz só o DELETE
        
        conn.commit();
    }
}
```
