## Spring Cloud Function

```java
@SpringBootApplication
public class LambdaApplication {

    public static void main(String[] args) {
        SpringApplication.run(LambdaApplication.class, args);
    }
    
    @Bean
    public Function<TarefaRequest, TarefaResponse> criarTarefa() {
        return request -> {
            // lógica
            return new TarefaResponse(request.nome(), true);
        };
    }
}
```

## Serverless YAML

```yaml
# serverless.yml
service: stayhard-api

provider:
  name: aws
  runtime: java17
  stage: ${opt:stage, 'dev'}
  region: us-east-1
  environment:
    TABLE_NAME: ${self:service}-${self:provider.stage}
  
functions:
  criarTarefa:
    handler: br.com.stayhard.LambdaApplication
    events:
      - http:
          path: /tarefas
          method: post
          cors: true
    memorySize: 512
    timeout: 30

resources:
  Resources:
    TarefasTable:
      Type: AWS::DynamoDB::Table
      Properties:
        TableName: ${self:provider.environment.TABLE_NAME}
        AttributeDefinitions:
          - AttributeName: id
            AttributeType: S
        KeySchema:
          - AttributeName: id
            KeyType: HASH
        BillingMode: PAY_PER_REQUEST
```
