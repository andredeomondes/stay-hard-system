# Dia 146 — IA: Prompt Engineering

## Conceito do Dia

**Prompt Engineering** é a arte de escrever prompts eficazes para obter melhores resultados dos LLMs.

## Técnicas

### 1. Few-Shot Learning

```java
String prompt = """
    Classifique a prioridade das tarefas:
    
    Exemplos:
    - "Reunião com cliente às 3pm" → ALTA
    - "Ligar para mãe" → MEDIA
    - "Organizar gaveta" → BAIXA
    
    Classifique:
    - "%s" →
    """.formatted(tarefa);
```

### 2. Chain of Thought

```java
String prompt = """
    Para cada tarefa, siga este raciocínio:
    
    1. A tarefa é urgente? (deadline próximo ou impacto imediato)
    2. A tarefa é importante? (alinhada com objetivos)
    3. Decisão:
       - Urgente + Importante → ALTA
       - Não urgente + Importante → MEDIA  
       - Não urgente + Não importante → BAIXA
    
    Analise: %s
    
    Raciocínio:
    """.formatted(tarefa);
```

### 3. Structured Output

```java
String prompt = """
    Analise a tarefa e responda neste formato EXATO:
    
    PRIORIDADE: [ALTA|MEDIA|BAIXA]
    JUSTIFICATIVA: [1-2 frases]
    CRITÉRIOS_USADOS: [lista de critérios]
    
    Tarefa: %s
    """.formatted(tarefa);
```

### 4. System + User + Assistant

```java
String systemPrompt = """
    Você é um coach de produtividade com 20 anos de experiência.
    Seu nome é Carlos.
    Você ajuda pessoas a serem mais produtivas com técnicas como Pomodoro e GTD.
    Seja direto e motivador.
    """;

String userPrompt = request.message();

// Historico da conversa
String assistantPrompt = """
    Assistista: Entendi! Vamos focar em uma coisa de cada vez.
    Usuário: Como melhorar minha produtividade?
    """;

Prompt prompt = Prompt.builder()
    .system(systemPrompt)
    .message(assistantPrompt)
    .user(userPrompt)
    .build();
```

## Anti-patterns

| Errado | Certo |
|--------|-------|
| "faça" genérico | "Crie 3 subtarefas" específico |
| Sem contexto | Com exemplo e formato |
| Múltiplas perguntas | Uma pergunta por vez |
