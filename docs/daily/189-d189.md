# Dia 189 — Eventos: CQRS e Event Sourcing

## Conceito do Dia

**CQRS** separa leitura de escrita. **Event Sourcing** armazena eventos, não estado.

---

## CQRS (Command Query Responsibility Segregation)

```
WRITE SIDE:
Tarefa -> Command -> TarefaCommandHandler -> Event Store

READ SIDE:
Query -> TarefaProjection -> Read Model (cache/tabela)
```

## Command

```java
public sealed interface TarefaCommand {
    
    record CriarTarefa(
        String nome,
        Categoria categoria,
        Prioridade prioridade
    ) implements TarefaCommand {}
    
    record ConcluirTarefa(Long id) implements TarefaCommand {}
    
    record AtualizarTarefa(
        Long id,
        String nome,
        Prioridade prioridade
    ) implements TarefaCommand {}
}
```

## Command Handler

```java
@Service
public class TarefaCommandHandler {
    
    private final EventStore eventStore;
    
    public void handle(TarefaCommand command) {
        List<Event> events = switch (command) {
            case CriarTarefa c -> List.of(
                new TarefaCriadaEvent(
                    UUID.randomUUID().toString(),
                    c.nome(),
                    c.categoria(),
                    c.prioridade(),
                    LocalDateTime.now()
                )
            );
            case ConcluirTarefa c -> List.of(
                new TarefaConcluidaEvent(c.id(), LocalDateTime.now())
            );
            case AtualizarTarefa a -> List.of(
                new TarefaAtualizadaEvent(
                    a.id(), a.nome(), a.prioridade(), LocalDateTime.now()
                )
            );
        };
        
        events.forEach(eventStore::save);
    }
}
```

## Projection (Read Model)

```java
@Service
public class TarefaProjection {
    
    @EventListener
    public void on(TarefaCriadaEvent event) {
        TarefaReadModel model = new TarefaReadModel();
        model.setId(event.id());
        model.setNome(event.nome());
        modelRepository.save(model);
    }
    
    @EventListener
    public void on(TarefaConcluidaEvent event) {
        Optional<TarefaReadModel> model = modelRepository.findById(event.id());
        model.ifPresent(m -> {
            m.setConcluida(true);
            modelRepository.save(m);
        });
    }
}
```
