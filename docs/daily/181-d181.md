# Dia 181 — Contribuição Open Source

## Conceito do Dia

Contribuir para projetos open source desenvolve habilidades e networking.

---

## Como Contribuir

```
1. Fork do repo
2. Clone localmente
3. Crie branch feature/bugfix
4. Faça mudanças
5. Teste
6. Commit e push
7. Abra Pull Request
```

## Como Encontrar Projetos

| Site | Tipo |
|------|------|
| Good First Issues | Issues para iniciantes |
| Up For Grabs | Tasks disponíveis |
| CodeTriage | Issues por repo |

## Boas Práticas

```bash
# Fork no GitHub
# Clone seu fork
git clone https://github.com/youruser/project.git

# Adicionar upstream
git remote add upstream https://github.com/original/project.git

# Atualizar
git fetch upstream
git merge upstream/main

# Criar branch
git checkout -b fix/typo-in-readme

# Fazer mudanças
git commit -m "fix: correct typo in README"

# Push
git push origin fix/typo-in-readme
```

## Pull Request Template
