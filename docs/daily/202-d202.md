## Como Participar

```markdown
## Contribuir para comunidade

1. **Ler antes de perguntar**
   - FAQ, documentação
   - Tópicos anteriores

2. **Perguntas de qualidade**
   ```
   ❌ "não funciona" (ruim)
   ✅ "Erro 500 ao chamar /api/tarefas após deploy no ECS.
   Já verifiquei logs do CloudWatch: [erro]." (bom)
   ```

3. **Responder dúvidas**
   - Ajude quem está começando
   - Compartilhe sua experiência

4. **Compartilhar conhecimento**
   - Artigos no Dev.to
   - Snippets úteis
   - Tutoriais
```
