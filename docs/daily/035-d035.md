## 💻 CHECKLIST DE CÓDIGO

### 📁 Exercício 1: try-catch-finally

```java
public String lerArquivo(String caminho) {
    BufferedReader reader = null;
    try {
        reader = new BufferedReader(new FileReader(caminho));
        return reader.readLine();
    } catch (FileNotFoundException e) {
        System.out.println("Arquivo não encontrado: " + caminho);
        return null;
    } catch (IOException e) {
        System.out.println("Erro ao ler: " + e.getMessage());
        return null;
    } finally {
        // Sempre executa, mesmo com return
        if (reader != null) {
            try { reader.close(); } catch (IOException e) {}
        }
    }
}
```

**Checklist:**
- [ ] try-catch
- [ ] múltiplos catch
- [ ] finally
- [ ] try-with-resources

### 📁 Exercício 2: try-with-resources

```java
// Automaticamente fecha recursos
public String lerArquivo(String caminho) {
    try (BufferedReader reader = new BufferedReader(
            new FileReader(caminho))) {
        return reader.readLine();
    } catch (IOException e) {
        System.out.println("Erro: " + e.getMessage());
        return null;
    }
    // Não precisa finally! Fecha automaticamente
}
```

### 📁 Exercício 3: throw e throws

```java
// Lançar exceção
public void sacar(double valor) {
    if (valor <= 0) {
        throw new IllegalArgumentException("Valor inválido");
    }
    if (valor > saldo) {
        throw new SaldoInsuficienteException("Saldo insuficiente");
    }
    saldo -= valor;
}

// Declarar que pode lançar
public void metodoPerigoso() throws IOException {
```
