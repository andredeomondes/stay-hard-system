## Classificador Simples

```java
@Service
public class PrioridadePredictor {
    
    private final ClassificationModel model;
    
    public PrioridadePredictor() throws Exception {
        // Carregar modelo pré-treinado
        model = ClassificationModel.load("prioridade-model.zip");
    }
    
    public Prioridade prever(TarefaFeatures features) {
        List<String> labels = model.predict(features.toArray());
        return Prioridade.valueOf(labels.get(0));
    }
}

public record TarefaFeatures(
    int categoriaCode,
    int horaCriacao,
    int diaSemana,
    int diaMes,
    int historicoConclusao
) {}
```

## Regressão Linear

```java
@Service
public class ProdutividadePredictor {
    
    public double preverTempoEstimado(Tarefa tarefa) {
        // Simplificado: base + coeficientes
        double base = 30; // minutos
        double complexidade = switch (tarefa.getPrioridade()) {
            case ALTA -> 1.5;
            case MEDIA -> 1.0;
            case BAIXA -> 0.5;
        };
        double categoria = switch (tarefa.getCategoria()) {
            case TRABALHO -> 1.2;
            case ESTUDO -> 1.1;
            default -> 1.0;
        };
        
        return base * complexidade * categoria;
    }
}
```
