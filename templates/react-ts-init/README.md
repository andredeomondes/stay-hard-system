# React + TypeScript Init Template

Template básico para iniciar projetos React com TypeScript.

## Como usar

1. Copie esta pasta para seu novo projeto
2. Renomeie no package.json
3. `npm install`
4. `npm run dev`

## Stack

- React 18+
- TypeScript
- Vite
- Axios
- React Router DOM
- Tailwind CSS

## Estrutura

```
src/
├── App.tsx
├── main.tsx
├── routes/
│   └── AppRoutes.tsx
├── pages/
├── components/
├── services/
│   └── api.ts
├── hooks/
├── context/
├── types/
└── styles/
    └── globals.css
```
