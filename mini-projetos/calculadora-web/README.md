# Calculadora Web

Calculadora responsiva com HTML + CSS + JavaScript.

## Stack
- HTML5 + CSS3
- JavaScript (ES6+)

## Features
- Operações: +, -, *, /
- Display que mostra resultado
- Botões para números e operações
- Design responsivo
- Keyboard support

## Como rodar
Abra `index.html` no navegador.
