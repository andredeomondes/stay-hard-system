# Weather App

Aplicação de clima que consome API pública.

## Stack
- HTML5 + CSS3
- JavaScript (ES6+, Fetch API)
- OpenWeatherMap API

## Features
- Buscar clima por cidade
- Mostrar temperatura, descrição, ícone
- Consumir API via fetch
- Tratamento de erros
- Interface responsiva

## Como rodar
1. Obtenha API key em https://openweathermap.org/api
2. Substitua `YOUR_API_KEY` no código
3. Abra `index.html` no navegador
