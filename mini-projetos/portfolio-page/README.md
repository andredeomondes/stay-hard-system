# Portfolio Page

Página de portfolio pessoal com HTML + CSS.

## Stack
- HTML5 (semântico)
- CSS3 (flexbox, grid, responsivo)

## Features
- Seção Sobre mim
- Seção Projetos (cards)
- Seção Contato
- Header com navegação
- Design responsivo (mobile-first)
- Deploy no GitHub Pages

## Como rodar
Abra `index.html` no navegador ou faça deploy no GitHub Pages.
